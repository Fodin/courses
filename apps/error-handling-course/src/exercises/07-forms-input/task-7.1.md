# Задание 7.1: Ошибки валидации

## Цель
Реализовать клиентскую валидацию формы с отображением ошибок под полями.

## Требования
1. Создайте форму с полями: name, email, age
2. Реализуйте функцию `validate()` с правилами:
   - name: обязательно, минимум 2 символа
   - email: обязательно, формат email
   - age: обязательно, число 18-120
3. Валидируйте при submit, убирайте ошибки при вводе (после первого submit)
4. Подсвечивайте ошибочные поля красной рамкой
5. Отображайте сообщения под каждым полем

## Чеклист
- [ ] 3 поля с разными правилами валидации
- [ ] Ошибки показываются при submit
- [ ] Ошибки убираются при исправлении
- [ ] Поля подсвечены красным при ошибке
- [ ] Тексты ошибок под полями
