# Задание 10.3: Type-Level Strings

## Цель

Реализовать продвинутые операции над строковыми литеральными типами: Split, Join, Replace, ReplaceAll, Trim.

## Требования

1. Реализуйте `Split<S, D>` -- разбивает строковый литерал по разделителю в кортеж строк
2. Реализуйте `Join<T, D>` -- объединяет кортеж строк через разделитель в один строковый литерал
3. Реализуйте `Replace<S, From, To>` -- заменяет первое вхождение подстроки
4. Реализуйте `ReplaceAll<S, From, To>` -- заменяет все вхождения подстроки
5. Реализуйте `TrimLeft<S>`, `TrimRight<S>` и `Trim<S>` для удаления пробелов
6. Для каждой type-level операции покажите runtime-аналог

## Чеклист

- [ ] `Split<"a-b-c", "-">` = `["a", "b", "c"]`
- [ ] `Split<"single", "-">` = `["single"]`
- [ ] `Join<["hello", "world"], "-">` = `"hello-world"`
- [ ] `Replace<"hello world", "world", "TS">` = `"hello TS"`
- [ ] `ReplaceAll<"a-b-c", "-", "_">` = `"a_b_c"`
- [ ] `Trim<"  hello  ">` = `"hello"`
- [ ] Split и Join являются обратными операциями: `Join<Split<S, D>, D>` = S
- [ ] Compile-time assertions подтверждают результаты

## Как проверить себя

1. Проверьте `Split<"", "-">` -- должен быть `[]`
2. Проверьте `Replace` с пустым `From` -- должен вернуть исходную строку
3. Убедитесь, что `Trim` обрабатывает табуляцию и переводы строк
