# Task 2.2: Run flags

## Objective
Learn the main flags of the `docker run` command.

## Requirements
Create a flag reference component with:

1. **Categorization** (mode, network, environment, resources, etc.)
2. For each flag: name, description, usage example
3. Show at least 10 flags

## Key flags
- `-d`, `-it`, `--rm`, `--name`
- `-p` (ports), `--network`
- `-e`, `--env-file`
- `-v`, `--memory`, `--cpus`, `--restart`

## Checklist
- [ ] Flags are organized by category
- [ ] Each one has a description and an example
- [ ] At least 10 flags are shown
