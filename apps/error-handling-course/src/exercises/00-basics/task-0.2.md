# Задание 0.2: Объект Error и его свойства

## Цель

Изучить структуру объекта `Error` и встроенные типы ошибок JavaScript.

## Требования

1. Создайте `Error` через `new Error('...')` и изучите его свойства:
   - `name` — имя типа ошибки
   - `message` — текст сообщения
   - `stack` — стек вызовов
2. Создайте экземпляры всех встроенных типов ошибок:
   - `Error`, `TypeError`, `RangeError`, `SyntaxError`, `ReferenceError`, `URIError`
3. Отобразите таблицу со свойствами каждого типа
4. Покажите полный стек вызовов для одной из ошибок

## Чеклист

- [ ] Объект Error создан и его свойства выведены
- [ ] Все встроенные типы ошибок созданы
- [ ] Таблица со свойствами отображается
- [ ] Стек вызовов показан

## Как проверить себя

1. Нажмите "Исследовать Error" — должны появиться name, message, stack
2. Нажмите "Создать разные ошибки" — таблица с 6 типами ошибок
3. Убедитесь, что каждый тип имеет правильное свойство `name`
