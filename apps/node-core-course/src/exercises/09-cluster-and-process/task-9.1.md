# Задание 9.1: Process Object

## Цель

Изучить глобальный объект `process`: переменные окружения, аргументы, память, потоки ввода-вывода и свойства процесса.

## Требования

1. Продемонстрируйте `process.env` — чтение переменных, типы значений (всегда string)
2. Покажите `process.argv` — структуру массива аргументов
3. Покажите `process.memoryUsage()` — все поля с объяснением (rss, heapTotal, heapUsed, external)
4. Продемонстрируйте `process.exit()` vs `process.exitCode` — разницу подходов
5. Покажите `process.stdin/stdout/stderr` и другие полезные свойства (pid, platform, uptime)

## Чеклист

- [ ] process.env показан с примерами и предупреждением о типах
- [ ] process.argv структура объяснена
- [ ] Все поля memoryUsage описаны с единицами измерения
- [ ] Разница exit() vs exitCode объяснена
- [ ] stdin/stdout/stderr описаны как streams

## Как проверить себя

1. Нажмите кнопку запуска
2. Убедитесь, что каждое свойство process имеет пример и объяснение
3. Проверьте, что memoryUsage показывает конкретные числа в МБ
