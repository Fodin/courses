# Задание 0.1: Первый try/catch

## Цель

Научиться использовать конструкцию `try/catch/finally` для обработки ошибок.

## Требования

1. Создайте функцию, которая парсит JSON-строку через `JSON.parse`
2. Оберните вызов в `try/catch`:
   - В `try` — парсинг и вывод результата
   - В `catch` — обработка ошибки `SyntaxError`
3. Добавьте блок `finally`, который логирует завершение операции
4. Создайте пример с `throw new Error(...)` для создания своей ошибки
5. Отображайте результаты на странице

## Чеклист

- [ ] `try/catch` корректно оборачивает `JSON.parse`
- [ ] Ошибка `SyntaxError` обрабатывается в `catch`
- [ ] Блок `finally` выполняется всегда
- [ ] Используется `throw new Error`
- [ ] Результаты отображаются на странице

## Как проверить себя

1. Нажмите кнопку запуска
2. Убедитесь, что успешный парсинг показывает данные
3. Убедитесь, что некорректный JSON показывает ошибку
4. Убедитесь, что `finally` выполняется в обоих случаях
