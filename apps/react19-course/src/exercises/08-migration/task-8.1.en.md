# Task 8.1: React 18 Application Audit

## Objective

Build an interactive tool for auditing deprecated React 18 patterns and their React 19 replacements.

## Requirements

1. Create a list of deprecated APIs with "before" and "after" code examples
2. Add filtering by category (Components, Refs, Context, ReactDOM)
3. Show the severity level for each pattern
4. Add a "Mark as found" button for an interactive audit
5. Show audit progress

## Checklist

- [ ] At least 7 deprecated patterns
- [ ] "Before" (React 18) and "after" (React 19) code for each
- [ ] Filtering by category
- [ ] Severity level for each pattern
- [ ] Audit progress counter

## How to verify

1. Filters work — the list updates
2. The "Mark" button marks the pattern
3. Progress updates
