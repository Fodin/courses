# Задание 5.3: Suspense при навигации

## Цель

Использовать Suspense и transition для плавной навигации между «страницами».

## Требования

1. Создайте несколько «страниц» (lazy-компонентов или компонентов с задержкой)
2. Навигация обёрнута в `startTransition`
3. Suspense показывает fallback при первой загрузке
4. При повторной навигации старая страница остаётся видимой (transition)

## Чеклист

- [ ] Несколько страниц с имитацией загрузки
- [ ] Навигация через `startTransition`
- [ ] Suspense fallback для первой загрузки
- [ ] Плавный переход между страницами

## Как проверить себя

1. Первая загрузка показывает spinner
2. Переключение между страницами — старая страница видна пока грузится новая
