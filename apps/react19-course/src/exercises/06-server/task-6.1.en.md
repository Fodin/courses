# Task 6.1: Server vs Client Components

## Objective

Learn to determine which components should be server-side and which should be client-side.

## Requirements

1. Create an interactive card game
2. Show cards with component descriptions
3. The user chooses: "Server" or "Client"
4. After answering, show the correct answer with an explanation

## Checklist

- [ ] At least 5 cards with components
- [ ] Selection buttons "Server" / "Client"
- [ ] Correct/incorrect answer is highlighted
- [ ] Explanation of why a component is server or client
- [ ] Correct answers counter

## How to verify

1. Read the component description
2. Choose the type — server or client
3. Check the explanation
