# Задание 4.4: Cleaning up reactions — Memory Leak Fix

## 🎯 Цель

Создать компонент с несколькими реакциями (autorun и reaction) и правильно утилизировать их все при размонтировании, чтобы предотвратить утечки памяти.

## 📋 Требования

1. Создайте класс `TimerStore` с observable полями:
   - `seconds: number` (по умолчанию `0`)
   - `isRunning: boolean` (по умолчанию `false`)
2. Добавьте методы `start()`, `stop()`, `tick()`, `reset()`
3. В компоненте внутри `useEffect` создайте:
   - **autorun**, который следит за `isRunning` и управляет `setInterval` для вызова `tick()` каждую секунду
   - **reaction**, который следит за `seconds` и логирует каждые 5 секунд
4. В cleanup-функции `useEffect` утилизируйте **все** реакции:
   - Вызовите disposer для autorun
   - Вызовите disposer для reaction
   - Очистите interval через `clearInterval`
5. Добавьте кнопки Start, Stop, Reset

## Чеклист

- [ ] `TimerStore` создан с `makeAutoObservable`
- [ ] autorun запускает/останавливает interval при изменении `isRunning`
- [ ] reaction логирует каждые 5 секунд (`seconds % 5 === 0`)
- [ ] **Все** disposer-ы вызываются в cleanup `useEffect`
- [ ] interval очищается в cleanup `useEffect`
- [ ] Компонент отображает предупреждение о важности cleanup

## 🔍 Как проверить себя

1. Нажмите Start — таймер начнёт считать секунды, в логе появится запись от autorun
2. Подождите 5 секунд — в логе появится запись от reaction
3. Нажмите Stop — таймер остановится, в логе появится запись об остановке
4. Нажмите Reset — секунды сбросятся в 0
5. Главная проверка: перейдите на другую вкладку (другой уровень) и обратно. Если cleanup работает правильно — таймер не продолжает работать в фоне, нет ошибок в консоли
