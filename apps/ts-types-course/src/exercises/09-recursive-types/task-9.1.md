# Задание 9.1: Recursive Data Structures

## Цель

Научиться моделировать рекурсивные структуры данных в TypeScript: JSON, деревья, связные списки, файловые системы и AST.

## Требования

1. Создайте тип `JsonValue` — рекурсивный union для представления произвольного JSON (string, number, boolean, null, массив, объект)
2. Реализуйте generic `TreeNode<T>` с полями `value` и `children` и функцию рекурсивного обхода `treeToString`
3. Создайте `LinkedList<T>` с `value` и `next` (или `null`) и функцию преобразования в массив `linkedListToArray`
4. Реализуйте `FileSystemEntry` с рекурсивными `children` и функцию подсчёта файлов `countFiles`
5. Создайте `Comment` с рекурсивными `replies` и функцию подсчёта `countComments`
6. Реализуйте AST через discriminated union `Expression` с типами `number`, `string`, `binary`, `unary` и функцию `evaluate`

## Чеклист

- [ ] `JsonValue` принимает произвольные вложенные JSON структуры
- [ ] `TreeNode<T>` корректно строит дерево произвольной глубины
- [ ] `LinkedList<T>` заканчивается `null` и конвертируется в массив
- [ ] `FileSystemEntry` различает файлы и директории, считает файлы рекурсивно
- [ ] `Comment` поддерживает вложенные ответы, `countComments` считает все
- [ ] `Expression` AST вычисляется рекурсивно через `evaluate`
- [ ] Результаты отображаются на странице

## Как проверить себя

1. Нажмите кнопку «Запустить»
2. Убедитесь, что дерево корректно выводится с отступами
3. Проверьте, что связный список `1 -> 2 -> 3` корректно отображается
4. Убедитесь, что `evaluate(1 + 2 * 3)` возвращает 7
