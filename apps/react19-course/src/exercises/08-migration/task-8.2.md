# Задание 8.2: Пошаговая миграция

## Цель

Создать интерактивный чеклист пошаговой миграции с React 18 на React 19.

## Требования

1. Создайте список шагов миграции (минимум 7)
2. Каждый шаг содержит: название, описание, команды (если есть), советы
3. Добавьте чекбоксы для отметки выполненных шагов
4. Покажите progress bar
5. При завершении всех шагов покажите поздравление

## Чеклист

- [ ] Шаги миграции в правильном порядке
- [ ] Чекбоксы для каждого шага
- [ ] Progress bar обновляется
- [ ] Команды терминала отображаются в стиле кода
- [ ] Поздравление при 100% прогрессе

## Как проверить себя

1. Отмечайте шаги — прогресс растёт
2. Отметьте все шаги — появится поздравление
