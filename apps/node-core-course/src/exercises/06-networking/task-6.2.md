# Задание 6.2: HTTP from Scratch

## Цель

Научиться создавать HTTP сервер на модуле `http` без фреймворков: обработка запросов, чтение body, маршрутизация и streaming response.

## Требования

1. Создайте HTTP сервер через `http.createServer()`
2. Покажите свойства `IncomingMessage`: method, url, headers
3. Продемонстрируйте чтение body через async iterator с ограничением размера
4. Реализуйте простую маршрутизацию (GET, POST, 404)
5. Покажите streaming response через `pipeline(readStream, res)`
6. Симулируйте несколько HTTP запросов/ответов

## Чеклист

- [ ] HTTP сервер создан и слушает порт
- [ ] IncomingMessage свойства показаны
- [ ] Чтение body с ограничением размера реализовано
- [ ] Маршрутизация GET/POST/404 работает
- [ ] Streaming response через pipeline показан
- [ ] Симуляция запросов корректна

## Как проверить себя

1. Нажмите "Запустить" и изучите симуляцию запросов
2. Убедитесь, что GET возвращает 200, POST возвращает 201, неизвестный путь -- 404
3. Проверьте, что тело POST-запроса корректно обрабатывается
4. Убедитесь, что streaming response объяснён
