# Задание 1.2: boolean и date

## Цель

Создать схемы для boolean и date типов, продемонстрировать валидацию чекбокса и даты.

## Требования

1. Создайте `boolSchema` — `yup.boolean().required().isTrue()` с кастомным сообщением для `isTrue`

2. Создайте `dateSchema` — `yup.date().required().min(new Date('2000-01-01'))` с кастомным сообщением

3. Для boolean используйте чекбокс (checkbox) — покажите, как `isTrue()` требует значения `true`

4. Для date используйте input type="date" — покажите, как Yup парсит строку-дату в объект Date

5. При успешной валидации отобразите значение и его тип

## Чеклист

- [ ] `boolSchema` создана с `isTrue()`
- [ ] `dateSchema` создана с `min()`
- [ ] Чекбокс в состоянии false вызывает ошибку `isTrue`
- [ ] Дата до 2000 года вызывает ошибку `min`
- [ ] Результат показывает значение и тип

## Как проверить себя

1. Не ставьте галочку и нажмите "Validate" — ошибка "Must be checked"
2. Поставьте галочку — успех, значение `true`
3. Введите дату `1999-12-31` — ошибка "Date must be after 2000"
4. Введите дату `2024-01-15` — успех, значение будет ISO-строкой Date
