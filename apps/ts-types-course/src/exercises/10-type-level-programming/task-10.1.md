# Задание 10.1: Type-Level Arithmetic

## Цель

Реализовать арифметические операции на уровне типов TypeScript, используя кодирование Пеано через длину кортежей.

## Требования

1. Реализуйте вспомогательный тип `BuildTuple<N>`, который создаёт кортеж длины N, заполненный `unknown`
2. Реализуйте тип `Add<A, B>`, который складывает два натуральных числа через конкатенацию кортежей
3. Реализуйте тип `Subtract<A, B>`, который вычитает B из A через деструктуризацию кортежа (A >= B)
4. Реализуйте тип `Multiply<A, B>`, который перемножает два числа через рекурсивное сложение
5. Продемонстрируйте каждую операцию runtime-аналогом, используя настоящие массивы
6. Добавьте compile-time проверки через присваивание литеральных значений результирующим типам

## Чеклист

- [ ] `BuildTuple<3>` создаёт `[unknown, unknown, unknown]`
- [ ] `Add<3, 4>` вычисляется в `7`
- [ ] `Subtract<10, 3>` вычисляется в `7`
- [ ] `Multiply<4, 5>` вычисляется в `20`
- [ ] Вложенные операции работают: `Add<Multiply<3, 3>, 1>` = `10`
- [ ] Runtime-демонстрации отображаются при нажатии кнопки "Запустить"
- [ ] Compile-time assertions проходят без ошибок

## Как проверить себя

1. Попробуйте `const x: Add<3, 4> = 8` -- должна быть ошибка компиляции (ожидается 7)
2. Проверьте `Subtract<3, 5>` -- должен вернуть `never` (отрицательные числа не поддерживаются)
3. Убедитесь, что `Multiply<0, 100>` корректно возвращает `0`
