# Задание 11.1: Exact Types

## Цель

Реализовать тип `Exact<T, Shape>`, который запрещает лишние свойства в объектах, и продемонстрировать проблему excess properties в TypeScript.

## Требования

1. Продемонстрируйте проблему: покажите, как непрямое присваивание позволяет лишние свойства
2. Реализуйте `Exact<T, Shape>` через `Exclude<keyof T, keyof Shape> extends never`
3. Реализуйте функцию `applyConfig<T>`, которая принимает только точное совпадение формы
4. Реализуйте `DeepExact<T, Shape>` для вложенных объектов
5. Покажите практические примеры: API-контракты, конфигурации

## Чеклист

- [ ] Продемонстрирована проблема с excess properties при непрямом присваивании
- [ ] `Exact<T, Shape>` корректно отклоняет объекты с лишними свойствами
- [ ] `applyConfig` принимает точное совпадение
- [ ] `applyConfig` отклоняет объекты с дополнительными полями (compile-time)
- [ ] Перечислены практические use-cases

## Как проверить себя

1. Создайте объект с лишним полем и передайте в `applyConfig` -- должна быть ошибка
2. Создайте объект с точной формой -- должен пройти
3. Попробуйте DeepExact с вложенным лишним полем
