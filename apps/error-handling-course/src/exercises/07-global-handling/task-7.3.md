# Задание 7.3: React Error Context

## Цель
Реализовать централизованную обработку ошибок через React Context.

## Требования
1. Создайте `ErrorContext` с функциями: addError, removeError, clearErrors
2. Создайте `ErrorProvider` компонент
3. Поддержите два уровня: `error` и `warning`
4. Warnings автоматически исчезают через 5 секунд
5. Создайте `ErrorDisplay` — показывает все текущие ошибки с кнопкой закрытия
6. Создайте компонент, демонстрирующий добавление ошибок

## Чеклист
- [ ] ErrorContext и ErrorProvider созданы
- [ ] addError/removeError/clearErrors работают
- [ ] Два уровня: error и warning
- [ ] Warnings исчезают через 5 секунд
- [ ] ErrorDisplay с кнопками закрытия
