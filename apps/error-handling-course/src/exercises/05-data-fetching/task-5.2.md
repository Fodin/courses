# Задание 5.2: API ошибки

## Цель
Создать класс ApiError и продемонстрировать типизированную обработку API-ошибок.

## Требования
1. Создайте класс `ApiError extends Error` с полями: `status`, `code`, `details`
2. Создайте функцию-симулятор API с разными ответами (200, 403, 404, 500)
3. Обработайте каждый тип ошибки через `instanceof ApiError`
4. Покажите детали ошибок (status, code, message, details)

## Чеклист
- [ ] `ApiError` класс с status, code, details
- [ ] Симулятор API с разными ответами
- [ ] Типизированная обработка через instanceof
- [ ] Детали ошибок отображаются
