# Task 9.3: OS Module

## Goal

Study the `os` module for system information: CPU, memory, platform, network interfaces.

## Requirements

1. Show `os.cpus()` — core count, models, speed
2. Demonstrate `os.totalmem()` / `os.freemem()` with usage calculation
3. Show platform information: platform, type, arch, release, hostname
4. Demonstrate `os.networkInterfaces()` — IP addresses
5. Show `os.constants` — signals and errno codes

## Checklist

- [ ] CPU information with core count
- [ ] Memory: total, free, used with percentages
- [ ] All key platform properties
- [ ] Network interfaces with IPv4/IPv6
- [ ] os.constants for signals and errno

## How to Verify

1. Click the run button
2. Verify CPU shows model and core count
3. Check memory is displayed in human-readable units (GB)
