# Задание 7.4: Мониторинг ошибок

## Цель
Создать систему мониторинга ошибок с детальными отчётами.

## Требования
1. Создайте класс `ErrorMonitor` с методом `report(error, extra?)`
2. Отчёт содержит: message, stack, url, timestamp, userAgent, extra
3. Симулируйте отправку на сервер (console.log)
4. Реализуйте подписку для React
5. Отобразите список отчётов с деталями

## Чеклист
- [ ] `ErrorMonitor` класс с report()
- [ ] Структура отчёта: message, stack, url, timestamp, userAgent
- [ ] Extra-данные для контекста ошибки
- [ ] Отчёты отображаются с деталями
