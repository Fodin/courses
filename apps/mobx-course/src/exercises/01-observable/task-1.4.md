# Задание 1.4: observable.ref

## Цель
Понять разницу между deep observable и observable.ref.

## Требования

- [ ] Создайте `DataStore` с массивом объектов `{id, name, value}`
- [ ] Используйте `observable.ref` для массива (отслеживание только замены ссылки)
- [ ] Добавьте метод `setItems` для замены массива целиком
- [ ] Добавьте computed `sortedItems` — отсортированный массив
- [ ] Покажите обе версии (оригинальный и отсортированный) в компоненте
- [ ] Добавьте кнопки для добавления/удаления элементов
