# Задание 2.1: email и url

## Цель

Создать схемы для валидации email и URL с помощью встроенных методов Yup.

## Требования

1. Создайте `emailSchema` — `yup.string().required().email()` с кастомными сообщениями

2. Создайте `urlSchema` — `yup.string().required().url()` с кастомными сообщениями

3. Для каждой схемы создайте отдельное поле ввода и кнопку "Validate"

4. При успешной валидации покажите результат в зелёном блоке

5. При ошибке покажите сообщение в красном блоке

## Чеклист

- [ ] `emailSchema` создана с `email()` и кастомным сообщением
- [ ] `urlSchema` создана с `url()` и кастомным сообщением
- [ ] Каждая схема валидируется отдельно
- [ ] Результаты отображаются в цветных блоках

## Как проверить себя

1. Email: "user@example.com" — успех
2. Email: "bad-email" — ошибка "Invalid email format"
3. Email: пустое — ошибка "required"
4. URL: "https://google.com" — успех
5. URL: "google.com" — ошибка (нет протокола)
6. URL: "not a url" — ошибка
