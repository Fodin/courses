# Task 8.4: Error Monitoring

## Goal
Create an error monitoring system with detailed reports.

## Requirements
1. Create an `ErrorMonitor` class with method `report(error, extra?)`
2. Report contains: message, stack, url, timestamp, userAgent, extra
3. Simulate sending to server (console.log)
4. Implement subscription for React
5. Display list of reports with details

## Checklist
- [ ] `ErrorMonitor` class with report()
- [ ] Report structure: message, stack, url, timestamp, userAgent
- [ ] Extra data for error context
- [ ] Reports displayed with details
