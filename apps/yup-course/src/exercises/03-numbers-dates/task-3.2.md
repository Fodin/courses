# Задание 3.2: integer и truncate

## Цель

Освоить валидацию целых чисел и трансформации `truncate()` / `round()`.

## Требования

1. Создайте `quantitySchema` — число, обязательное, положительное, целое (`integer()`)

2. Создайте `truncateSchema` — число с трансформацией `truncate()`

3. Создайте `roundSchema` — число с трансформацией `round('ceil')`

4. Реализуйте валидацию целого числа с отображением результата

5. Реализуйте демонстрацию трансформаций: покажите оригинал и результат каждой трансформации

## Чеклист

- [ ] `quantitySchema` проверяет, что число целое через `integer()`
- [ ] `truncateSchema` отбрасывает дробную часть через `truncate()`
- [ ] `roundSchema` округляет вверх через `round('ceil')`
- [ ] Дробное число (3.7) не проходит валидацию `integer()`
- [ ] Трансформации корректно отображают результат

## Как проверить себя

1. Quantity: 5 — успех
2. Quantity: 3.7 — ошибка "Must be a whole number"
3. Quantity: -2 — ошибка (не положительное)
4. Transform 3.7: truncate() = 3, round("ceil") = 4
5. Transform -2.3: truncate() = -2, round("ceil") = -2
