# Задание 10.2: Streaming Uploads

## 🎯 Цель

Освоить потоковую загрузку больших файлов через busboy: stream pipe в файл, отслеживание прогресса, обработка лимитов и пост-обработка изображений через sharp.

## Требования

1. Настройте busboy с headers и limits для обработки multipart-запроса
2. Реализуйте потоковую запись: `stream.pipe(createWriteStream(path))` без буферизации в RAM
3. Отслеживайте прогресс загрузки через события `data` и Content-Length
4. Обработайте превышение лимита: событие `limit`, удаление частично загруженного файла
5. Покажите пост-обработку через sharp: resize в несколько размеров, конвертация в WebP

## Чеклист

- [ ] Busboy обрабатывает multipart через pipe (req.pipe(bb))
- [ ] Файл записывается потоково без буферизации в RAM
- [ ] Прогресс отслеживается в процентах
- [ ] Превышение лимита удаляет частичный файл
- [ ] Sharp генерирует thumb/medium/large версии

## Как проверить себя

Нажмите "Запустить" и убедитесь, что: файл загружается потоково с минимальным потреблением RAM, прогресс отображается, лимиты работают, изображения обрабатываются sharp.
