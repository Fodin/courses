# Задание 3.3: useFormStatus

## Цель

Использовать `useFormStatus` для отображения состояния отправки формы в дочерних компонентах.

## Требования

1. Импортируйте `useFormStatus` из `'react-dom'`
2. Создайте компонент `SubmitButton`, который использует `useFormStatus`
3. Во время отправки кнопка должна быть disabled и показывать «Отправка...»
4. `SubmitButton` должен быть **дочерним** элементом `<form>`

## Чеклист

- [ ] `useFormStatus` импортирован из `'react-dom'`
- [ ] `SubmitButton` — отдельный компонент внутри формы
- [ ] `pending` из `useFormStatus` управляет disabled и текстом кнопки
- [ ] Форма использует `action` для отправки
- [ ] Кнопка меняет состояние во время отправки

## Как проверить себя

1. Нажмите кнопку отправки
2. Кнопка становится disabled и показывает «Отправка...»
3. После завершения кнопка возвращается в исходное состояние
