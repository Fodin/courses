# Задание 4.1: Базовый useOptimistic

## Цель

Использовать `useOptimistic` для мгновенного отображения изменения до завершения серверного запроса.

## Требования

1. Создайте состояние `liked` (boolean) через `useState`
2. Используйте `useOptimistic(liked, (_, newValue) => newValue)` для оптимистичного значения
3. Создайте async-функцию `toggleLike`, которая мгновенно обновляет UI и потом вызывает «сервер»
4. Кнопка лайка должна мгновенно переключаться

## Чеклист

- [ ] `useOptimistic` импортирован из `'react'`
- [ ] Оптимистичное значение отображается мгновенно
- [ ] Серверный запрос имитируется через `setTimeout`
- [ ] После ответа сервера состояние синхронизируется

## Как проверить себя

1. Нажмите кнопку лайка — она мгновенно меняется
2. Через 1-2 секунды изменение «подтверждается» сервером
