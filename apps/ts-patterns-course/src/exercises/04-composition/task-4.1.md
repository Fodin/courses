# Задание 4.1: Pipe и Compose

## Цель

Реализовать типобезопасные функции `pipe` и `compose` для построения цепочек преобразований данных.

## Требования

1. Создайте функцию `pipe` с перегрузками для 2, 3 и 4 функций
   - `pipe(fn1, fn2)` — результат fn1 передаётся в fn2
   - Каждая перегрузка должна проверять совместимость типов между функциями
2. Создайте функцию `compose` с перегрузками — аналогично, но справа налево
3. Создайте вспомогательные функции для демонстрации:
   - `trim(s: string): string`
   - `toUpperCase(s: string): string`
   - `addExclamation(s: string): string`
   - `getLength(s: string): number`
   - `double(n: number): number`
   - `toFixed(n: number): string`
4. Продемонстрируйте цепочки с разными типами: `string → string → number → string`
5. Покажите, что несовместимые типы вызывают ошибку компиляции

## Чеклист

- [ ] `pipe` имеет перегрузки для 2–4 функций
- [ ] `compose` имеет перегрузки для 2–4 функций
- [ ] Типы проверяются между шагами (number не передать в string-функцию)
- [ ] Вспомогательные функции созданы и работают
- [ ] Демонстрация выводит результаты на экран

## Как проверить себя

- `pipe(trim, toUpperCase)("  hello  ")` должен вернуть `"HELLO"`
- `pipe(trim, getLength, double)("  hi  ")` должен вернуть `4`
- `compose(toUpperCase, trim)("  hello  ")` должен вернуть `"HELLO"`
