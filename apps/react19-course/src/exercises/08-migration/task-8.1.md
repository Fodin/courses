# Задание 8.1: Аудит React 18 приложения

## Цель

Создать интерактивный инструмент для аудита deprecated паттернов React 18 и их замен в React 19.

## Требования

1. Создайте список deprecated API с примерами кода «до» и «после»
2. Добавьте фильтрацию по категориям (Components, Refs, Context, ReactDOM)
3. Для каждого паттерна покажите severity (критичность)
4. Добавьте кнопку «Отметить как найденное» для интерактивного аудита
5. Покажите прогресс аудита

## Чеклист

- [ ] Минимум 7 deprecated паттернов
- [ ] Код «до» (React 18) и «после» (React 19) для каждого
- [ ] Фильтрация по категориям
- [ ] Уровень severity для каждого паттерна
- [ ] Счётчик прогресса аудита

## Как проверить себя

1. Фильтры работают — список обновляется
2. Кнопка «Отметить» помечает паттерн
3. Прогресс обновляется
