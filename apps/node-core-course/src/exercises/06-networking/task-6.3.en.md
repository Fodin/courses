# Task 6.3: URL & DNS

## Goal

Master the WHATWG URL API, URLSearchParams, and the `dns` module for domain name resolution.

## Requirements

1. Show URL parsing via `new URL()` with all fields
2. Demonstrate `URLSearchParams`: get, set, append, toString, iteration
3. Show programmatic URL construction
4. Demonstrate `dns.lookup()` vs `dns.resolve()` with comparison table
5. Show different DNS record types: A, AAAA, MX, TXT, NS, CNAME
6. Simulate DNS resolution for multiple record types

## Checklist

- [ ] URL parsed into all components (protocol, hostname, port, pathname, search, hash)
- [ ] URLSearchParams with get/set/append operations shown
- [ ] Programmatic URL construction demonstrated
- [ ] dns.lookup() vs dns.resolve() comparison table present
- [ ] DNS record types shown
- [ ] DNS resolution simulation works

## How to verify

1. Click "Run" and study the URL parsing
2. Verify all URL components are shown correctly
3. Check URLSearchParams correctly builds query strings
4. Verify the difference between lookup and resolve is explained
