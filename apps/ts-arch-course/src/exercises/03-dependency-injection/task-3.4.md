# Задание 3.4: Auto-Wiring

## 🎯 Цель

Реализовать автоматическое разрешение зависимостей: при resolve сервиса контейнер рекурсивно разрешает все его зависимости, обнаруживая циклы.

## Требования

1. Создайте `AutoWireContainer` с методом `register(name, dependencies, factory)`
2. dependencies — массив строк с именами зависимостей, которые будут переданы в factory
3. `resolve()` рекурсивно разрешает все зависимости перед созданием сервиса
4. Реализуйте обнаружение циклических зависимостей с информативным сообщением
5. Добавьте `getResolutionOrder()` — порядок, в котором зависимости будут разрешены
6. Добавьте `getDependencyGraph()` — граф всех зависимостей
7. Зарегистрируйте минимум 6 сервисов с глубиной зависимостей >= 3

## Чеклист

- [ ] Зависимости автоматически разрешаются рекурсивно
- [ ] Циклические зависимости обнаруживаются с путём цикла в ошибке
- [ ] `getResolutionOrder()` возвращает правильный topological order
- [ ] Один сервис создаётся только один раз (кеширование)
- [ ] Граф зависимостей показывает все связи

## Как проверить себя

Создайте циклическую зависимость (A -> B -> C -> A) и убедитесь, что ошибка содержит полный путь. Проверьте resolution order — зависимости должны идти до зависящих от них.
