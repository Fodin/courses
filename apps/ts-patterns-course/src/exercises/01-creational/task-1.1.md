# Задание 1.1: Factory Method

## Цель

Реализовать паттерн Factory Method для создания разных типов уведомлений через единый интерфейс.

## Требования

1. Создайте интерфейс `Notification` с методами:
   - `send(message: string): string` — отправка уведомления
   - `format(message: string): string` — форматирование сообщения
2. Реализуйте три класса, имплементирующих `Notification`:
   - `EmailNotification` — форматирует в HTML (`<html><body>...</body></html>`)
   - `SMSNotification` — обрезает до 160 символов
   - `PushNotification` — добавляет эмодзи-префикс
3. Создайте union-тип `NotificationType = 'email' | 'sms' | 'push'`
4. Создайте фабричную функцию `createNotification(type: NotificationType): Notification`
5. Продемонстрируйте создание и использование всех типов уведомлений

## Чеклист

- [ ] Интерфейс `Notification` определён с `send` и `format`
- [ ] Три класса реализуют интерфейс
- [ ] Union-тип `NotificationType` ограничивает допустимые значения
- [ ] Фабричная функция возвращает интерфейс, а не конкретный класс
- [ ] Демонстрация — кнопка выводит результаты отправки и форматирования

## Как проверить себя

1. Нажмите кнопку запуска
2. Убедитесь, что каждый тип уведомления форматирует сообщение по-своему
3. Убедитесь, что `createNotification('email')` возвращает `Notification`, а не `EmailNotification`
4. Попробуйте передать невалидный тип — TypeScript должен показать ошибку
