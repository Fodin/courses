Курс yup-course: 9 уровней (0-8), 33 задания. Библиотека Yup v1.6.

**Why:** Курс по библиотеке валидации Yup с практическими заданиями.

**How to apply:** При создании контента для оставшихся уровней сверяться с exercisesConfig.tsx и translations.ts.

Прогресс по уровням:
- Level 0 (00-intro): контент готов (2 задания: первая схема, обработка ошибок)
- Level 1 (01-primitives): контент готов (4 задания: string/number, boolean/date, required/nullable, default/defined)
- Level 2 (02-strings): контент готов (4 задания: email/url, min/max/length, matches, trim/lowercase/uppercase)
- Level 3 (03-numbers-dates): контент готов (4 задания: min/max/positive/negative, integer/truncate, date validation, date ranges with ref)
- Level 4 (04-objects): контент готов (4 задания: object/shape, nested objects, pick/omit/partial, noUnknown/strict)
- Level 5 (05-arrays-tuples): контент готов (3 задания: array.of(), min/max/length, tuple)
- Level 6 (06-conditional): не создан (4 задания)
- Level 7 (07-custom-transforms): не создан (4 задания)
- Level 8 (08-advanced): не создан (4 задания)
