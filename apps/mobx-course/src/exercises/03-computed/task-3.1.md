# Задание 3.1: Первый computed

## Цель
Создать стор корзины с computed-значением для итоговой цены.

## Требования

- [ ] Создайте `CartStore` с массивом `items: {name, price, qty}[]`
- [ ] Добавьте computed геттер `totalPrice` — сумма `price * qty`
- [ ] Добавьте методы `setQty`, `addItem`, `removeItem`
- [ ] Покажите таблицу товаров с кнопками +/- для количества
- [ ] Покажите итоговую сумму (обновляется автоматически)
