# Задание 5.3: Восстановление после ошибки

## Цель
Реализовать Error Boundary с возможностью восстановления (retry).

## Требования
1. Создайте `RecoverableErrorBoundary` с кнопкой "Попробовать снова"
2. При нажатии — сбросьте `hasError` и увеличьте `resetKey`
3. Используйте `key={resetKey}` на обёртке children для пересоздания дерева
4. Создайте `RandomFailure` — компонент, падающий с 50% вероятностью
5. Покажите, что после нажатия кнопки компонент пересоздаётся

## Чеклист
- [ ] `RecoverableErrorBoundary` с кнопкой reset
- [ ] `resetKey` + `key` для пересоздания дерева
- [ ] `RandomFailure` компонент
- [ ] Восстановление работает при нажатии
