# Задание 2.4: docker exec

## Цель
Научиться выполнять команды внутри запущенного контейнера.

## Требования
Создайте компонент с примерами `docker exec`:

1. Покажите **типичные сценарии** отладки (файлы, сеть, окружение, БД)
2. Объясните ключевые флаги: `-it`, `-u`, `-w`, `-e`
3. Добавьте примеры для разных ситуаций

## Проверьте себя

- [ ] Минимум 8 примеров с описаниями
- [ ] Объяснены ключевые флаги
