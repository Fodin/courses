# Задание 2.2: min, max, length

## Цель

Создать схемы с ограничениями длины строки.

## Требования

1. Создайте `usernameSchema` — `yup.string().required().min(3).max(20)` для имени пользователя

2. Создайте `pinSchema` — `yup.string().required().length(4)` для PIN-кода

3. Для каждой схемы покажите текущую длину введённой строки

4. При успехе покажите значение и его длину

5. При ошибке покажите сообщение

## Чеклист

- [ ] `usernameSchema` создана с `min(3)` и `max(20)`
- [ ] `pinSchema` создана с `length(4)`
- [ ] Текущая длина строки отображается под полем ввода
- [ ] Результаты валидации показываются в цветных блоках

## Как проверить себя

1. Username: "ab" — ошибка "At least 3 characters"
2. Username: "john" — успех, length: 4
3. Username: длинная строка из 25 символов — ошибка "At most 20 characters"
4. PIN: "123" — ошибка "exactly 4 characters"
5. PIN: "1234" — успех
6. PIN: "12345" — ошибка
