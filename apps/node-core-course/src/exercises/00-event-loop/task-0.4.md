# Задание 0.4: Real-world Ordering Puzzle

## Цель

Применить знания о фазах Event Loop для решения сложной задачи на порядок выполнения асинхронного кода.

## Требования

1. Проанализируйте сложный фрагмент кода с mix из setTimeout, setImmediate, nextTick и Promise
2. Пошагово разберите каждую фазу Event Loop
3. Покажите, какие колбэки ставятся в очередь на каждом шаге
4. Выведите финальный порядок выполнения с объяснением
5. Сформулируйте ключевые выводы

## Чеклист

- [ ] Предоставлен сложный пример с 5+ асинхронными вызовами
- [ ] Каждый шаг Event Loop разобран с указанием текущей фазы
- [ ] Показано состояние очередей на каждом шаге
- [ ] Финальный порядок выведен и объяснён
- [ ] Сформулированы ключевые выводы

## Как проверить себя

1. Откройте задание и изучите пример кода
2. Попробуйте предсказать порядок самостоятельно
3. Нажмите "Запустить" и сравните с пошаговым разбором
4. Убедитесь, что понимаете каждый шаг
