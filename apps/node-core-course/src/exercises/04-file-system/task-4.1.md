# Задание 4.1: Sync/Async Read/Write

## Цель

Разобраться в трёх стилях API модуля `fs` (callback, sync, promises) и научиться выполнять базовые операции чтения и записи файлов.

## Требования

1. Продемонстрируйте три стиля работы с fs: callback, synchronous, promises
2. Покажите операции `readFile`, `writeFile`, `appendFile` с объяснением каждой
3. Объясните разницу между чтением с encoding и без (Buffer vs string)
4. Продемонстрируйте работу с file flags (`w`, `a`, `wx`, `r+`)
5. Покажите использование file handles (`fs.open`) с корректным закрытием через `try/finally`
6. Симулируйте виртуальную файловую систему для демонстрации операций

## Чеклист

- [ ] Показаны все 3 стиля API с примерами кода
- [ ] Продемонстрированы readFile, writeFile, appendFile
- [ ] Объяснена разница Buffer vs string (encoding)
- [ ] Показаны file flags и их назначение
- [ ] Продемонстрированы file handles с try/finally
- [ ] Работает симуляция с виртуальной FS

## Как проверить себя

1. Нажмите "Запустить" и изучите вывод
2. Убедитесь, что для каждого стиля API показан соответствующий синтаксис
3. Проверьте, что симуляция корректно демонстрирует writeFile и readFile
4. Убедитесь, что appendFile добавляет строки, а не перезаписывает
