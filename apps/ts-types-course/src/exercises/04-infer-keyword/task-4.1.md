# Задание 4.1: Infer в возвращаемых типах

## Цель

Научиться использовать `infer` для извлечения типов возвращаемых значений из функций, промисов и вложенных структур.

## Требования

1. Создайте тип `MyReturnType<T>`, аналогичный встроенному `ReturnType` — он должен извлекать тип возвращаемого значения функции
2. Создайте тип `UnwrapPromise<T>`, который извлекает тип из `Promise<T>`, а для не-промисов возвращает исходный тип
3. Создайте тип `DeepUnwrapPromise<T>`, рекурсивно разворачивающий вложенные промисы (`Promise<Promise<Promise<number>>>` -> `number`)
4. Создайте тип `ArrayElement<T>`, извлекающий тип элемента массива
5. Продемонстрируйте комбинацию типов: извлеките тип данных из async-функции, которая возвращает `Promise<массив>`

## Чеклист

- [ ] `MyReturnType` корректно извлекает тип возвращаемого значения
- [ ] `UnwrapPromise` работает и с промисами, и с обычными типами
- [ ] `DeepUnwrapPromise` рекурсивно разворачивает вложенные промисы
- [ ] `ArrayElement` извлекает тип элемента из массива
- [ ] Комбинация типов корректно работает с async-функциями
- [ ] Результаты отображаются на странице

## Как проверить себя

1. Нажмите кнопку «Запустить»
2. Убедитесь, что `MyReturnType` возвращает объектный тип для функции, возвращающей объект
3. Убедитесь, что `DeepUnwrapPromise` корректно разворачивает 3+ уровня вложенности
4. Проверьте, что `ArrayElement<string[]>` дает `string`
