# Задание 6.2: use client директива

## Цель

Понять, когда и зачем добавлять `'use client'` директиву.

## Требования

1. Покажите примеры кода компонентов
2. Пользователь решает, нужна ли директива `'use client'`
3. Объясните правила: useState/useEffect → клиентский, обработчики событий → клиентский
4. Покажите, что серверные компоненты не нуждаются в директиве

## Чеклист

- [ ] Примеры компонентов с и без `'use client'`
- [ ] Интерактивный выбор для каждого примера
- [ ] Объяснения правил
- [ ] Визуальная подсветка директивы в коде

## Как проверить себя

1. Для каждого примера определите нужна ли директива
2. Проверьте правильность через подсказки
