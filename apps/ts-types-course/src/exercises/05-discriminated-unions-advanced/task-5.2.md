# Задание 5.2: Полиморфные обработчики

## Цель

Научиться создавать типобезопасные dispatch-таблицы для обработки вариантов размеченных объединений.

## Требования

1. Создайте размеченное объединение `AppEvent` с минимум 4 типами событий, каждое с уникальным `payload`
2. Создайте utility-тип `EventPayload<T>`, извлекающий payload по типу события с помощью `Extract`
3. Реализуйте типобезопасную dispatch-таблицу `EventHandlers`, где каждый обработчик получает правильный тип payload
4. Создайте generic-функцию `dispatch<T>(type, payload)`, которая вызывает соответствующий обработчик
5. Продемонстрируйте middleware-паттерн: функцию, принимающую `AppEvent` и `next` callback

## Чеклист

- [ ] `AppEvent` содержит минимум 4 варианта с разными payload
- [ ] `EventPayload<T>` корректно извлекает payload через `Extract`
- [ ] Dispatch-таблица типизирована через mapped type
- [ ] `dispatch` функция generic и проверяет соответствие типа и payload
- [ ] Middleware корректно типизирован

## Как проверить себя

1. Нажмите кнопку «Запустить»
2. Убедитесь, что каждый dispatch вызывает правильный обработчик
3. Попробуйте вызвать `dispatch('USER_LOGIN', { url: '/' })` — должна быть ошибка типов
4. Убедитесь, что middleware обрабатывает событие и вызывает `next`
