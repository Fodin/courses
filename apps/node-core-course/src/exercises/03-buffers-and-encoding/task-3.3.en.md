# Task 3.3: Binary Protocol Parsing

## Goal

Learn to parse and create binary data in TLV (Type-Length-Value) format.

## Requirements

1. Implement encoding a message into binary format [type][length][payload]
2. Implement decoding a binary message
3. Create multiple TLV entries of different types (string, uint32, boolean)
4. Show TLV structure parsing
5. List Buffer methods for reading/writing numbers

## Checklist

- [ ] Message encoding to binary format
- [ ] Decoding from binary format
- [ ] TLV with multiple data types
- [ ] TLV parsing works correctly
- [ ] readUInt/writeUInt methods listed

## How to verify

1. Run the demo
2. Verify encoded message decodes correctly
3. Ensure all TLV entries parse correctly
