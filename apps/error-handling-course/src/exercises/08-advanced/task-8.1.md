# Задание 8.1: Функциональная обработка ошибок

## Цель
Реализовать функциональные утилиты для работы с Result типом.

## Требования
1. Реализуйте `map(result, fn)` — трансформация значения
2. Реализуйте `flatMap(result, fn)` — цепочка Result-операций
3. Реализуйте `mapError(result, fn)` — трансформация ошибки
4. Реализуйте `unwrapOr(result, default)` — значение по умолчанию
5. Создайте pipeline: parse → validate → sqrt → double
6. Визуально покажите результат и ошибки

## Чеклист
- [ ] `map`, `flatMap`, `mapError`, `unwrapOr` реализованы
- [ ] Pipeline из 4 операций
- [ ] Корректный результат при валидном вводе
- [ ] Информативная ошибка при невалидном вводе
