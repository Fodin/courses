# Задание 3.1: form action

## Цель

Использовать новый React 19 prop `action` у формы для обработки отправки без `onSubmit` и `preventDefault`.

## Требования

1. Создайте async-функцию, принимающую `FormData`
2. Передайте её в `<form action={fn}>`
3. Извлеките данные из `FormData` (name, email)
4. После «отправки» покажите результат

## Чеклист

- [ ] Форма использует `action={asyncFunction}` вместо `onSubmit`
- [ ] Функция принимает `FormData`
- [ ] Данные извлекаются через `formData.get()`
- [ ] Результат отображается после отправки

## Как проверить себя

1. Заполните поля формы
2. Нажмите «Отправить»
3. Данные отображаются на странице
