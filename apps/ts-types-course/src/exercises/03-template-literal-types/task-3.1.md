# Задание 3.1: Template Literal Basics

## Цель

Научиться создавать union-типы из template literal types, понять декартово произведение при комбинировании union-типов и применить шаблонные строковые типы к реальным задачам.

## Требования

1. Создайте тип `ColorVariant` = `${Shade}-${Color}`, где Shade = 'light' | 'dark', Color = 'red' | 'green' | 'blue'. Создайте массив со всеми вариантами
2. Создайте тип `CSSValue` = `${number}${CSSUnit}` для типобезопасных CSS-значений. Реализуйте функцию `setCSSValue`
3. Создайте тип `EventHandler` = `on${Capitalize<DOMEvent>}` для имён обработчиков событий
4. Создайте тип `ApiRoute` = `/api/${Version}/${Resource}` для маршрутов API. Сгенерируйте все возможные комбинации
5. Создайте тип `LogEntry` = `[${HttpMethod}] ${StatusCode}` для строк лога. Покажите количество комбинаций

## Чеклист

- [ ] `ColorVariant` содержит все 6 комбинаций (2 * 3)
- [ ] `CSSValue` принимает '100px', '1.5rem', '50vh', но не '100' без единицы
- [ ] `EventHandler` правильно capitalize-ит имена событий ('click' -> 'onClick')
- [ ] `ApiRoute` содержит все комбинации версий и ресурсов
- [ ] Продемонстрировано декартово произведение и его потенциальный рост

## Как проверить себя

1. Попробуйте присвоить невалидную строку типу CSSValue -- ошибка
2. Подсчитайте количество вариантов LogEntry вручную и сравните
3. Убедитесь, что все варианты ColorVariant присутствуют в массиве
