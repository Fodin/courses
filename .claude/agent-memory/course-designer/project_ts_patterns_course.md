Курс ts-patterns-course: 9 уровней (0-8), 33 задания.

**Why:** Курс по TypeScript-паттернам проектирования с практическими заданиями.

**How to apply:** При создании контента для оставшихся уровней сверяться с exercisesConfig.tsx и translations.ts.

Прогресс по уровням:
- Level 0 (type-contracts): контент готов (3 задания)
- Level 1 (creational): контент готов (4 задания)
- Level 2 (structural): контент готов (4 задания)
- Level 3 (behavioral): контент готов (4 задания)
- Level 4 (composition): контент готов (3 задания)
- Level 5 (data-management): контент готов (4 задания: Repository, Unit of Work, CQRS, Event Sourcing)
- Level 6 (error-results): не создан (3 задания)
- Level 7 (modular-architecture): не создан (4 задания)
- Level 8 (advanced-ts): контент готов (4 задания: Type-safe Builder, Phantom Types, Type-level State Machine, Effect Pattern)
