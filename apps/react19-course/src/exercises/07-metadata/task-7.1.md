# Задание 7.1: Document metadata

## Цель

Рендерить `<title>` и `<meta>` прямо в компоненте, используя автоматический hoisting React 19.

## Требования

1. Создайте несколько «страниц» с разными заголовками
2. Каждая страница рендерит `<title>` и `<meta name="description">` в JSX
3. Добавьте навигацию между страницами
4. Заголовок документа (вкладки браузера) должен меняться при переключении

## Чеклист

- [ ] `<title>` рендерится в JSX компонента
- [ ] `<meta>` рендерится в JSX компонента
- [ ] Переключение страниц обновляет заголовок вкладки
- [ ] Минимум 3 страницы с разными заголовками

## Как проверить себя

1. Переключите страницу — заголовок вкладки браузера изменился
2. Откройте DevTools → Elements → `<head>` — там появились ваши meta-теги
