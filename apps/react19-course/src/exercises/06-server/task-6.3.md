# Задание 6.3: Server Actions

## Цель

Понять концепцию Server Actions — серверных функций, вызываемых с клиента.

## Требования

1. Создайте симуляцию Server Action (async-функция с задержкой)
2. Покажите как Server Action используется в form `action`
3. Покажите как Server Action вызывается напрямую
4. Объясните `'use server'` директиву

## Чеклист

- [ ] Форма с `action` использует серверную функцию
- [ ] Прямой вызов Server Action по кнопке
- [ ] Показан код с `'use server'` директивой
- [ ] Объяснена сериализация данных

## Как проверить себя

1. Отправьте форму — данные обрабатываются «на сервере»
2. Нажмите кнопку прямого вызова — Server Action выполняется
