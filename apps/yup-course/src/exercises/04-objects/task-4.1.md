# Задание 4.1: object и shape

## Цель

Создать объектную схему для валидации данных пользователя с несколькими полями разных типов.

## Требования

1. Создайте `userSchema` с полями: `name` (строка, обязательная, min 2), `age` (число, обязательное, positive, integer), `email` (строка, обязательная, email)

2. Реализуйте форму с тремя полями ввода

3. Валидируйте данные с `abortEarly: false` для показа всех ошибок

4. При успехе покажите JSON валидированного объекта в зелёном блоке

5. При ошибке покажите все ошибки в красном блоке

## Чеклист

- [ ] `userSchema` создана через `yup.object()` с тремя полями
- [ ] Каждое поле имеет правильный тип и кастомные сообщения
- [ ] `abortEarly: false` передаётся в `validate()`
- [ ] Пустые поля обрабатываются корректно (пустая строка -> undefined для required)
- [ ] Результат отображается в `<pre>` блоке

## Как проверить себя

1. Все поля заполнены корректно — показывает JSON объекта
2. Все поля пусты — показывает 3 ошибки (name, age, email)
3. Имя "A" — ошибка "Name too short"
4. Age: 0 — ошибка (не положительное)
5. Email: "bad" — ошибка "Invalid email"
