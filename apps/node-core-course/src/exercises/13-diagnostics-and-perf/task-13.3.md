# Задание 13.3: Zlib Compression

## Цель

Освоить модуль `zlib` для сжатия данных: gzip/gunzip, deflate/inflate, brotli, потоковое сжатие через pipeline и HTTP-сжатие ответов.

## Требования

1. Продемонстрируйте синхронное сжатие/распаковку через gzipSync/gunzipSync
2. Покажите deflate и brotli как альтернативные алгоритмы
3. Приведите сравнение алгоритмов по степени сжатия и скорости
4. Реализуйте потоковое сжатие файлов через pipeline
5. Покажите HTTP-сжатие ответов с определением Accept-Encoding
6. Продемонстрируйте параметры сжатия (level, BROTLI_PARAM_QUALITY)

## Чеклист

- [ ] gzip/gunzip синхронное сжатие с показом ratio
- [ ] deflate и brotli как альтернативы
- [ ] Таблица сравнения алгоритмов
- [ ] Потоковое сжатие через pipeline
- [ ] HTTP-сжатие с Content-Encoding
- [ ] Параметры качества сжатия

## Как проверить себя

1. Нажмите "Запустить" — должны отобразиться все алгоритмы сжатия
2. Убедитесь, что показан коэффициент сжатия
3. Проверьте, что потоковое сжатие использует pipeline
4. Убедитесь, что HTTP-сжатие проверяет Accept-Encoding
