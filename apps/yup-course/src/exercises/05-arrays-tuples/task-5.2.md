# Задание 5.2: min, max, length для массивов

## Цель

Освоить ограничения длины массива: `min()`, `max()`, `length()`.

## Требования

1. Создайте `rolesSchema` — массив строк, обязательный, от 1 до 3 элементов

2. Создайте `teamSchema` — массив строк, обязательный, ровно 5 элементов

3. При успехе покажите количество элементов и сам массив

4. Парсите ввод по запятой

## Чеклист

- [ ] `rolesSchema` использует `min(1)` и `max(3)` с кастомными сообщениями
- [ ] `teamSchema` использует `length(5)` с кастомным сообщением
- [ ] Оба массива `required()`
- [ ] При успехе показывается count и JSON массива

## Как проверить себя

1. Roles: "admin, editor" — успех (2 roles)
2. Roles: "" — ошибка required
3. Roles: "a, b, c, d" — ошибка "Maximum 3 roles"
4. Team: "Alice, Bob, Charlie, Dave, Eve" — успех (5 members)
5. Team: "Alice, Bob" — ошибка "must have exactly 5"
