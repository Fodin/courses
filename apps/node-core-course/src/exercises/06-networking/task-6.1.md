# Задание 6.1: TCP with net

## Цель

Освоить модуль `net` для создания TCP серверов и клиентов, понять события сокета и свойства соединения.

## Требования

1. Покажите создание TCP сервера через `net.createServer()`
2. Продемонстрируйте TCP клиент через `net.createConnection()`
3. Покажите все события сокета: `connect`, `data`, `end`, `close`, `error`, `timeout`, `drain`
4. Продемонстрируйте свойства сокета: `remoteAddress`, `remotePort`, `bytesRead`, `bytesWritten`
5. Покажите серверные опции: `allowHalfOpen`, `keepAlive`, `maxConnections`
6. Симулируйте TCP echo server с обменом сообщениями

## Чеклист

- [ ] TCP server создан с обработкой входящих соединений
- [ ] TCP client подключается и отправляет данные
- [ ] Все события сокета перечислены и объяснены
- [ ] Свойства сокета показаны
- [ ] Серверные опции продемонстрированы
- [ ] Симуляция echo сервера работает

## Как проверить себя

1. Нажмите "Запустить" и изучите логи сервера и клиента
2. Убедитесь, что сервер отвечает echo на каждое сообщение
3. Проверьте, что показаны bytesRead и bytesWritten
4. Убедитесь, что объяснены основные события сокета
