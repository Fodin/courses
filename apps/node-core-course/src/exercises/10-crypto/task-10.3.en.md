# Task 10.3: Asymmetric Crypto

## Goal

Master asymmetric cryptography: key pair generation (generateKeyPairSync), digital signatures (sign/verify), and public key encryption (publicEncrypt/privateDecrypt).

## Requirements

1. Demonstrate RSA key pair generation with encoding settings
2. Show EC (Elliptic Curve) key generation
3. Implement simulation of creating and verifying a digital signature
4. Demonstrate encryption with public key and decryption with private key
5. Show the hybrid encryption concept (RSA + AES)

## Checklist

- [ ] RSA key pair generated with PEM encoding
- [ ] EC keys shown as an alternative
- [ ] Signature created with private key and verified with public key
- [ ] Encryption/decryption with asymmetric keys demonstrated
- [ ] RSA data size limitation mentioned
- [ ] Hybrid encryption idea shown

## How to verify

1. Click "Run" — the entire key management process should be displayed
2. Verify the signature is checked correctly
3. Check that the ~214 byte limit for RSA-2048 is mentioned
4. Verify hybrid encryption is explained
