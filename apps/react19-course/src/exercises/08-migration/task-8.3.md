# Задание 8.3: Финальный рефакторинг

## Цель

Создать компонент, демонстрирующий использование ВСЕХ основных API React 19 вместе.

## Требования

1. Используйте `use(Context)` для темы
2. Используйте `use(Promise)` с Suspense для загрузки данных
3. Используйте `useOptimistic` для мгновенного обновления
4. Используйте `useActionState` для формы добавления
5. Используйте `<title>` для метаданных
6. ref как обычный prop (без forwardRef)

## Чеклист

- [ ] `use(Context)` читает тему
- [ ] `use(Promise)` загружает данные
- [ ] `<Suspense>` показывает loading
- [ ] `useOptimistic` мгновенно обновляет UI
- [ ] `useActionState` обрабатывает форму
- [ ] `<title>` рендерится в компоненте
- [ ] Все API работают вместе

## Как проверить себя

1. Данные загружаются через Suspense
2. Переключение темы работает
3. Toggle задачи мгновенно (оптимистично) обновляет UI
4. Форма добавления работает с pending-состоянием
