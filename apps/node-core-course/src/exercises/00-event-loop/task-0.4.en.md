# Task 0.4: Real-world Ordering Puzzle

## Goal

Apply Event Loop phase knowledge to solve a complex async code ordering challenge.

## Requirements

1. Analyze a complex code snippet mixing setTimeout, setImmediate, nextTick, and Promise
2. Walk through each Event Loop phase step by step
3. Show which callbacks are queued at each step
4. Output the final execution order with explanations
5. Formulate key takeaways

## Checklist

- [ ] Complex example with 5+ async calls provided
- [ ] Each Event Loop step analyzed with current phase indicated
- [ ] Queue state shown at each step
- [ ] Final order output and explained
- [ ] Key takeaways formulated

## How to verify

1. Open the task and study the code example
2. Try to predict the order yourself
3. Click "Run" and compare with the step-by-step analysis
4. Make sure you understand each step
