# Задание 5.3: Loading/Error/Success

## Цель
Реализовать паттерн состояний загрузки с discriminated union.

## Требования
1. Определите тип `FetchState<T>` с 4 вариантами: idle, loading, success, error
2. Создайте хук `useFetch<T>(fetchFn)`, возвращающий `{ state, execute }`
3. Создайте функцию загрузки с 30% шансом ошибки
4. Отобразите все 4 состояния визуально:
   - idle: подсказка
   - loading: индикатор
   - error: сообщение + кнопка "Повторить"
   - success: данные

## Чеклист
- [ ] `FetchState<T>` определён
- [ ] `useFetch` хук реализован
- [ ] Все 4 состояния визуально отображаются
- [ ] Кнопка "Повторить" при ошибке
