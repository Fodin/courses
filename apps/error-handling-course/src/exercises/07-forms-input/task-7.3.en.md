# Task 7.3: Error Display

## Goal
Implement three error display patterns: inline, summary, and toast.

## Requirements
1. **Inline error** — under the specific field
2. **Error summary** — block above the form with list of all errors
3. **Toast notification** — temporary message that disappears after 3 seconds
4. Summary should use `role="alert"`
5. Show all 3 patterns when clicking buttons

## Checklist
- [ ] Inline error under field
- [ ] Error summary above form
- [ ] Toast notification with auto-hide
- [ ] `role="alert"` on summary
