# Задание 2.1: Basic EventEmitter

## Цель

Реализовать упрощённый EventEmitter и продемонстрировать основные методы API.

## Требования

1. Создайте класс SimpleEventEmitter с методами on, off, emit, removeAllListeners
2. Продемонстрируйте подписку и генерацию событий
3. Покажите удаление слушателей (off) и removeAllListeners
4. Выведите listenerCount и eventNames
5. Перечислите полный API Node.js EventEmitter

## Чеклист

- [ ] SimpleEventEmitter реализован с on, off, emit
- [ ] Демонстрация подписки и генерации работает
- [ ] Удаление слушателей корректно
- [ ] Показаны listenerCount и eventNames
- [ ] Перечислен полный API

## Как проверить себя

1. Запустите демонстрацию
2. Проверьте, что события корректно добавляются и удаляются
3. Сравните реализацию с полным API Node.js EventEmitter
