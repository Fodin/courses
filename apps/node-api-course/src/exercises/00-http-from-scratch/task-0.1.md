# Задание 0.1: http.createServer

## 🎯 Цель

Продемонстрировать работу `http.createServer`, объектов `IncomingMessage` и `ServerResponse`, а также правильное использование статус-кодов и заголовков.

## Требования

1. Создайте HTTP-сервер с помощью `http.createServer`
2. Обработайте объект `req` (IncomingMessage): выведите `method`, `url`, `httpVersion`, `headers`
3. Настройте объект `res` (ServerResponse): установите статус-код 200, заголовки `Content-Type: application/json` и `X-Request-Id`
4. Реализуйте вывод таблицы основных HTTP статус-кодов (200, 201, 204, 400, 401, 403, 404, 500)
5. Покажите полный цикл запрос-ответ с форматированием HTTP-сообщений

## Чеклист

- [ ] Сервер создан через `http.createServer` с callback `(req, res)`
- [ ] Свойства `req.method`, `req.url`, `req.headers` корректно выводятся
- [ ] `res.writeHead` вызывается со статус-кодом и заголовками
- [ ] Все основные статус-коды (2xx, 4xx, 5xx) перечислены с описанием
- [ ] Показан полный цикл: входящий запрос -> исходящий ответ
- [ ] Сервер запускается через `server.listen(port, callback)`

## Как проверить себя

Нажмите кнопку "Запустить" и убедитесь, что в выводе отображаются: структура объектов request/response, таблица статус-кодов и полный HTTP-цикл запрос-ответ.
