# Task 1.4: observable.ref

## Goal
Understand the difference between deep observable and observable.ref.

## Requirements

- [ ] Create `DataStore` with array of `{id, name, value}` objects
- [ ] Use `observable.ref` for the array (track only reference replacement)
- [ ] Add `setItems` method to replace the entire array
- [ ] Add computed `sortedItems` — sorted array
- [ ] Display both versions (original and sorted) in component
- [ ] Add buttons for adding/removing elements
