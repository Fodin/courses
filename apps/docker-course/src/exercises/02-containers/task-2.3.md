# Задание 2.3: Жизненный цикл контейнера

## Цель
Понять состояния контейнера и переходы между ними.

## Требования
Создайте компонент, показывающий:

1. **Диаграмму состояний**: Created → Running → Paused → Exited → Removed
2. Для каждого состояния — описание и команды для перехода
3. **Разницу** между `docker stop` (SIGTERM) и `docker kill` (SIGKILL)
4. Основные команды управления (`docker ps`, `start`, `stop`, `restart`, `rm`, `prune`)

## Проверьте себя
- [ ] Показаны все 5 состояний
- [ ] Для каждого состояния есть описание
- [ ] Объяснена разница SIGTERM vs SIGKILL
