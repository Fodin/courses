# Задание 2.2: use(Context)

## Цель

Заменить `useContext` на новый `use()` для чтения контекста, используя упрощённый синтаксис провайдера React 19.

## Требования

1. Создайте `ThemeContext` с помощью `createContext`
2. Создайте дочерний компонент, который читает тему через `use(ThemeContext)`
3. Используйте новый синтаксис провайдера: `<ThemeContext value={...}>` вместо `<ThemeContext.Provider>`
4. Добавьте кнопку переключения темы (light/dark)
5. Дочерний компонент должен менять стили в зависимости от темы

## Чеклист

- [ ] `createContext` создаёт контекст темы
- [ ] `use(ThemeContext)` используется вместо `useContext(ThemeContext)`
- [ ] Провайдер использует `<ThemeContext value={...}>` (React 19 синтаксис)
- [ ] Кнопка переключает тему
- [ ] Стили компонента меняются при смене темы

## Как проверить себя

1. Компонент отображается со светлой темой
2. Нажатие кнопки переключает на тёмную тему и обратно
