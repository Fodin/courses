# Задание 0.2: Microtasks vs Macrotasks

## Цель

Понять разницу между микротасками и макротасками, научиться предсказывать их порядок выполнения.

## Требования

1. Перечислите, какие API создают микротаски, а какие — макротаски
2. Продемонстрируйте ключевое правило: все микротаски дренируются после каждой макротаски
3. Покажите пример с вложенными Promise и setTimeout
4. Реализуйте живую демонстрацию в браузере с реальным `Promise.then`, `queueMicrotask` и `setTimeout`

## Чеклист

- [ ] Перечислены микротаски: Promise.then, process.nextTick, queueMicrotask
- [ ] Перечислены макротаски: setTimeout, setInterval, setImmediate, I/O
- [ ] Продемонстрировано правило приоритета микротасок
- [ ] Показан пример с вложенными цепочками Promise
- [ ] Есть живая демонстрация в браузере

## Как проверить себя

1. Откройте задание и нажмите "Запустить"
2. Убедитесь, что живая демонстрация показывает правильный порядок
3. Проверьте, что sync выполняется первым, затем микротаски, затем макротаски
