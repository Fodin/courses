# Задание 4.3: Откат при ошибке

## Цель

Показать, что оптимистичное обновление автоматически откатывается при ошибке сервера.

## Требования

1. Создайте оптимистичное обновление (лайк или добавление в список)
2. Имитируйте случайный сбой сервера (50% вероятность)
3. При ошибке оптимистичное значение должно откатиться
4. Покажите уведомление об ошибке

## Чеклист

- [ ] Оптимистичное обновление применяется мгновенно
- [ ] При ошибке сервера UI откатывается к предыдущему состоянию
- [ ] Пользователь видит уведомление об ошибке
- [ ] Можно попробовать снова

## Как проверить себя

1. Нажмите кнопку несколько раз
2. Иногда сервер «падает» и изменение откатывается
3. При успехе изменение сохраняется
