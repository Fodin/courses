# Задание 5.2: Writable Streams

## Цель

Освоить Writable Streams: запись данных, обработку переполнения буфера (`drain`), завершение записи (`end`/`finish`) и батчинг (`cork`/`uncork`).

## Требования

1. Продемонстрируйте `write()` и его возвращаемое значение (true/false)
2. Покажите обработку события `drain` при переполнении буфера
3. Продемонстрируйте `end()` для завершения записи и событие `finish`
4. Покажите `cork()`/`uncork()` для батчинга записей
5. Симулируйте запись с отслеживанием заполнения буфера
6. Покажите все ключевые события: `drain`, `finish`, `error`, `close`, `pipe`

## Чеклист

- [ ] write() с проверкой возвращаемого значения показан
- [ ] Событие drain обработано корректно
- [ ] end() и finish продемонстрированы
- [ ] cork()/uncork() показаны с объяснением
- [ ] Симуляция буфера работает с визуализацией
- [ ] Все ключевые события перечислены

## Как проверить себя

1. Нажмите "Запустить" и изучите заполнение буфера
2. Убедитесь, что write() возвращает false при переполнении
3. Проверьте, что drain сбрасывает буфер
4. Убедитесь, что cork/uncork объяснены через batch-оптимизацию
