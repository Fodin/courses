# Task 7.3: React Error Context

## Goal
Implement centralized error handling through React Context.

## Requirements
1. Create `ErrorContext` with functions: addError, removeError, clearErrors
2. Create `ErrorProvider` component
3. Support two levels: `error` and `warning`
4. Warnings automatically disappear after 5 seconds
5. Create `ErrorDisplay` — shows all current errors with close button
6. Create a component demonstrating error adding

## Checklist
- [ ] ErrorContext and ErrorProvider created
- [ ] addError/removeError/clearErrors work
- [ ] Two levels: error and warning
- [ ] Warnings disappear after 5 seconds
- [ ] ErrorDisplay with close buttons
