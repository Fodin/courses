# Задание 11.3: Test Database

## 🎯 Цель

Освоить управление тестовой БД: global setup/teardown, очистка между тестами, factories для генерации данных, паттерны для CI.

## Требования

1. Настройте global setup: создание тестовой БД, запуск миграций
2. Настройте global teardown: удаление тестовой БД
3. Реализуйте beforeEach: очистка таблиц в правильном порядке (FK constraints)
4. Создайте factory-функции: генерация уникальных данных (randomUUID для email/name)
5. Покажите seedTestData для создания связанных сущностей (admin, user, post)

## Чеклист

- [ ] Global setup создаёт тестовую БД и запускает миграции
- [ ] Global teardown удаляет тестовую БД
- [ ] beforeEach очищает таблицы с учётом FK constraints
- [ ] Factory-функции генерируют уникальные данные
- [ ] Тесты используют seedTestData и независимы друг от друга

## Как проверить себя

Нажмите "Запустить" и убедитесь, что: тестовая БД создаётся и удаляется, таблицы очищаются между тестами, factories генерируют уникальные данные, тесты независимы.
