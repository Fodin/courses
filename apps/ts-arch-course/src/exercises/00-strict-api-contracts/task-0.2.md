# Задание 0.2: Маппинг ответов

## 🎯 Цель

Реализовать типобезопасную систему маппинга API-ответов (DTO) в доменные типы с поддержкой композиции маперов.

## Требования

1. Создайте DTO-интерфейсы (snake_case) и соответствующие доменные интерфейсы (camelCase) для минимум 2 сущностей
2. Реализуйте тип `Mapper<TFrom, TTo>` и фабричную функцию `createMapper()`
3. Реализуйте функцию `mapArray()` — композитор, создающий маппер для массивов из маппера для одного элемента
4. Маперы должны трансформировать:
   - snake_case -> camelCase
   - Строковые даты -> объекты `Date`
   - Центы -> доллары (числовые трансформации)
   - Вложенные объекты
5. Продемонстрируйте работу маперов на реальных данных

## Чеклист

- [ ] DTO и доменные типы отражают реальную разницу API/фронтенд
- [ ] `createMapper` возвращает функцию с правильной типизацией `(TFrom) => TTo`
- [ ] `mapArray(mapUser)` автоматически выводит тип `Mapper<ApiUserDTO[], DomainUser[]>`
- [ ] Трансформации включают конверсию дат и числовых форматов
- [ ] Вложенные объекты маппятся корректно

## Как проверить себя

Проверьте, что маппер не позволяет пропустить обязательное поле доменного типа. Попробуйте убрать одно поле из объекта-результата — TypeScript должен показать ошибку.
