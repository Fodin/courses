# Задание 5.1: observer HOC

## 🎯 Цель

Научиться оборачивать React-компоненты в `observer` из `mobx-react-lite`, чтобы они автоматически реагировали на изменения observable-состояния.

## 📋 Требования

1. Создайте класс `TimerStore` с `makeAutoObservable`:
   - Observable поле `seconds: number` (начальное значение `0`)
   - Приватное поле `intervalId` для хранения идентификатора интервала
2. Добавьте action-методы:
   - `start()` — запускает `setInterval` с шагом 1 секунда (если таймер ещё не запущен)
   - `stop()` — останавливает интервал через `clearInterval`
   - `reset()` — останавливает таймер и сбрасывает `seconds` в `0`
   - `tick()` — увеличивает `seconds` на 1 (вызывается из `setInterval`)
3. Добавьте computed геттер `formatted` — возвращает строку в формате `MM:SS` (с ведущими нулями)
4. Создайте экземпляр стора на уровне модуля
5. Оберните компонент в `observer` и отобразите:
   - Отформатированное время крупным моноширинным шрифтом
   - Три кнопки: Start, Stop, Reset

```typescript
class TimerStore {
  seconds = 0
  private intervalId: ReturnType<typeof setInterval> | null = null

  get formatted(): string // "MM:SS"

  start(): void
  stop(): void
  reset(): void
  tick(): void
}
```

## Чеклист

- [ ] `TimerStore` создан с `makeAutoObservable`
- [ ] `start()` запускает интервал, повторный вызов не создаёт дублирующий интервал
- [ ] `stop()` останавливает интервал
- [ ] `reset()` останавливает и сбрасывает счётчик
- [ ] Computed `formatted` возвращает время в формате `MM:SS`
- [ ] Компонент обёрнут в `observer` и обновляется автоматически

## 🔍 Как проверить себя

1. Нажмите Start — таймер начнёт отсчёт, отображение обновляется каждую секунду
2. Нажмите Stop — отсчёт остановится, время зафиксируется
3. Нажмите Start снова — отсчёт продолжится с того же значения
4. Нажмите Reset — таймер сбросится на `00:00`
5. Убедитесь, что формат всегда `MM:SS` (например, `01:05`, а не `1:5`)
