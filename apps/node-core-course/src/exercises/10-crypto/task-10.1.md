# Задание 10.1: Hashing & HMAC

## Цель

Научиться использовать модуль `crypto` для хеширования данных (createHash), создания HMAC-подписей (createHmac) и генерации криптографически стойких случайных значений (randomBytes, randomUUID).

## Требования

1. Реализуйте симуляцию хеширования строки разными алгоритмами (MD5, SHA-256, SHA-512) с отображением результатов
2. Покажите множественный вызов `update()` для инкрементального хеширования
3. Продемонстрируйте создание HMAC с секретным ключом
4. Покажите безопасное сравнение через `timingSafeEqual`
5. Продемонстрируйте генерацию случайных данных: `randomBytes`, `randomUUID`, `randomInt`
6. Выведите список доступных алгоритмов хеширования

## Чеклист

- [ ] Показано хеширование тремя алгоритмами с результатами в hex
- [ ] Показано потоковое хеширование через множественный update
- [ ] Создан HMAC с демонстрацией проверки подписи
- [ ] Объяснена важность timingSafeEqual
- [ ] Продемонстрированы randomBytes, randomUUID, randomInt
- [ ] Компонент использует `useState` и кнопку запуска

## Как проверить себя

1. Нажмите "Запустить" — должны отобразиться хеши для разных алгоритмов
2. Убедитесь, что HMAC создан с секретным ключом
3. Проверьте, что объяснено, почему нельзя сравнивать хеши через `===`
4. Убедитесь, что показана генерация случайных данных
