## Section headings
- Цель → Objective
- Требования → Requirements
- Подсказки → Hints
- Чеклист → Checklist
- Как проверить себя → How to verify
- Проверьте себя → Checklist (when used as a header for checklist items)
- Уровень → Level
- Задание X.Y → Task X.Y
- Итоги → Summary

## Docker-specific terms
- Образ → image (or "Docker image")
- Контейнер → container
- Том → volume
- Слой → layer
- Реестр → registry
- Сборка → build
- Контекст сборки → build context
- Точка входа → entry point (or ENTRYPOINT when referring to the instruction)
- Записываемый слой → writable layer
- Базовый образ → base image
- Многоэтапная сборка → multi-stage build
- Эфемерный → ephemeral
- Непривилегированный пользователь → unprivileged user
- Проброс портов → port mapping / port publishing
- Висячий образ → dangling image

## ASCII diagrams in README files
- Translate only Russian text labels inside diagrams
- Leave box-drawing characters and code unchanged
- "Контейнеры" inside diagram → "Containers"
- "Гипервизор" → "Hypervisor"
- "Хост-операционная система" → "Host Operating System"
- "Физическое железо" → "Physical Hardware"
