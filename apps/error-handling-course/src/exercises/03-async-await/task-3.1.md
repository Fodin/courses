# Задание 3.1: async/await обработка

## Цель
Научиться обрабатывать ошибки в async/await коде через try/catch.

## Требования
1. Создайте функцию `simulateApi(shouldFail)`, возвращающую Promise
2. Оберните `await` в `try/catch/finally`
3. Покажите успешный и неуспешный запрос
4. Продемонстрируйте, что в цепочке последовательных `await` ошибка прерывает выполнение
5. Добавьте индикатор загрузки

## Чеклист
- [ ] `simulateApi` реализована
- [ ] `try/catch/finally` оборачивает `await`
- [ ] Показан успех и ошибка
- [ ] Цепочка `await` прерывается при ошибке
- [ ] Индикатор загрузки работает
