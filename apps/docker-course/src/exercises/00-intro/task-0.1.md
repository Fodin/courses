# Задание 0.1: Контейнеры vs виртуальные машины

## Цель

Разобраться в ключевых различиях между контейнерами и виртуальными машинами, научиться определять, какой подход лучше подходит для конкретных сценариев.

## Требования

1. Создайте сравнительную таблицу контейнеров и VM с как минимум 6 характеристиками
2. Для каждой характеристики укажите значение/описание для контейнеров и для VM
3. Используйте визуальные индикаторы (иконки/цвета) для обозначения преимуществ и недостатков
4. Добавьте секцию "Когда использовать что" с конкретными сценариями
5. Реализуйте интерактивность: кнопка для показа/скрытия подробного объяснения каждой характеристики

## Чеклист

- [ ] Таблица содержит минимум 6 строк с характеристиками
- [ ] Указаны значения для контейнеров и VM в каждой строке
- [ ] Есть визуальное выделение преимуществ (кто лучше по каждому критерию)
- [ ] Присутствует секция со сценариями использования
- [ ] Кнопка показа/скрытия подробностей работает корректно
- [ ] Компонент использует `useState` для управления состоянием

## Как проверить себя

1. Откройте задание в браузере
2. Убедитесь, что таблица отображается с правильными данными
3. Проверьте, что кнопка раскрытия подробностей переключает видимость
4. Сравните свою таблицу с решением -- все ключевые различия должны быть покрыты
