# Задание 4.3: Directory Operations

## Цель

Научиться выполнять операции с директориями: чтение содержимого, создание, удаление и рекурсивный обход дерева файлов.

## Требования

1. Продемонстрируйте `readdir` с опцией `withFileTypes` для определения типа записи
2. Покажите рекурсивный `readdir` (Node.js 18.17+)
3. Продемонстрируйте `mkdir` с `{ recursive: true }` для создания вложенных директорий
4. Покажите `rm` с `{ recursive: true, force: true }` для удаления деревьев
5. Реализуйте визуализацию дерева директорий
6. Покажите паттерн рекурсивного обхода с async generator

## Чеклист

- [ ] readdir с withFileTypes продемонстрирован
- [ ] Рекурсивный readdir показан
- [ ] mkdir с recursive создаёт вложенные директории
- [ ] rm с recursive удаляет деревья
- [ ] Визуализация дерева работает корректно
- [ ] Паттерн async generator для обхода показан

## Как проверить себя

1. Нажмите "Запустить" и изучите дерево директорий
2. Убедитесь, что дерево отображается с правильными символами (├──, └──)
3. Проверьте, что показаны API для создания, чтения и удаления
4. Убедитесь, что паттерн walkDir реализован корректно
