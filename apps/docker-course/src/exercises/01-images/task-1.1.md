# Задание 1.1: docker pull и реестры

## Цель

Разобраться с командой `docker pull` и различными реестрами образов.

## Требования

Создайте компонент, который отображает:

1. **Список реестров** — как минимум 3 реестра (Docker Hub, GitHub Container Registry, и один на ваш выбор)
2. Для каждого реестра покажите:
   - Название и краткое описание
   - Примеры команд `docker pull` с объяснениями
3. **Формат имени образа**: объясните схему `[реестр/][пространство/]имя[:тег][@sha256:дайджест]`

## Подсказки

- Docker Hub — реестр по умолчанию, его можно не указывать: `docker pull nginx` = `docker pull docker.io/library/nginx`
- `docker pull nginx:1.25-alpine` — конкретная версия
- `docker pull ghcr.io/owner/image:tag` — образ из GitHub Container Registry
- Дайджест (`@sha256:...`) гарантирует воспроизводимость
