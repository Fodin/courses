# Задание 1.1: Маршруты и Router

## 🎯 Цель

Освоить маршрутизацию в Express: методы `app.get/post/put/delete`, модульный `express.Router()`, параметры маршрута и query-параметры.

## Требования

1. Определите маршруты для CRUD операций с помощью `app.get()`, `app.post()`, `app.put()`, `app.patch()`, `app.delete()`
2. Покажите работу с параметрами маршрута `req.params` (например, `/api/users/:userId/posts/:postId`)
3. Покажите работу с query-параметрами `req.query` (например, `?page=2&limit=10`)
4. Создайте модульный роутер через `express.Router()` и подключите его через `app.use('/api/users', router)`
5. Продемонстрируйте, как Router объединяет маршруты в итоговую таблицу

## Чеклист

- [ ] Определены маршруты для всех CRUD-методов (GET, POST, PUT, PATCH, DELETE)
- [ ] `req.params` корректно извлекает динамические сегменты URL
- [ ] `req.query` корректно извлекает query-параметры
- [ ] Router вынесен в отдельный модуль и подключён через `app.use`
- [ ] Показано, как Router добавляет префикс к маршрутам

## Как проверить себя

Нажмите "Запустить" и убедитесь, что все тестовые запросы сопоставляются с правильными маршрутами, параметры извлекаются корректно, и Router показывает итоговые пути.
