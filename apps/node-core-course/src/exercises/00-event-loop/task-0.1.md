# Задание 0.1: Event Loop Phases

## Цель

Разобраться в фазах Event Loop Node.js и научиться предсказывать порядок выполнения `setTimeout`, `setImmediate` и `process.nextTick`.

## Требования

1. Создайте визуальную демонстрацию фаз Event Loop
2. Покажите диаграмму 6 фаз Event Loop с описанием каждой
3. Продемонстрируйте код с `setTimeout`, `setImmediate`, `process.nextTick` и ожидаемый порядок выполнения
4. Покажите разницу в поведении при вызове из main module и из I/O callback
5. Реализуйте кнопку "Запустить" для пошагового отображения результатов

## Чеклист

- [ ] Показана диаграмма фаз Event Loop
- [ ] Описаны все 6 фаз с примерами API для каждой
- [ ] Продемонстрирован порядок: sync → nextTick → Promise → setTimeout → setImmediate
- [ ] Объяснено поведение внутри I/O callback
- [ ] Компонент использует `useState` для управления состоянием

## Как проверить себя

1. Откройте задание в браузере
2. Нажмите "Запустить" и изучите порядок выполнения
3. Убедитесь, что объяснение соответствует реальному поведению Node.js
4. Проверьте, что упомянуто различие main module vs I/O callback
