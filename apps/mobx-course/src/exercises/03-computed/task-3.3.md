# Задание 3.3: Кеширование computed

## Цель
Убедиться, что computed кешируется и не пересчитывается без необходимости.

## Требования

- [ ] Создайте `ProductFilterStore` с массивом продуктов, фильтром по категории и рейтингу
- [ ] Добавьте computed `filteredAndSorted` с логированием в консоль
- [ ] Добавьте счетчик вычислений `computeCount`
- [ ] Добавьте кнопку "Force re-render" — при нажатии computed НЕ должен пересчитываться
- [ ] Убедитесь: пересчет происходит только при изменении фильтров
