# Задание 1.2: ECMAScript Modules (ESM)

## Цель

Изучить синтаксис и особенности ESM: import/export, top-level await, live bindings, import.meta.

## Требования

1. Покажите все варианты import/export синтаксиса
2. Объясните включение ESM: package.json "type": "module" и .mjs
3. Продемонстрируйте live bindings в сравнении с CJS copies
4. Покажите top-level await
5. Объясните замену __filename/__dirname через import.meta.url

## Чеклист

- [ ] Показан полный синтаксис import/export
- [ ] Объяснены способы включения ESM
- [ ] Продемонстрированы live bindings
- [ ] Показан top-level await
- [ ] Объяснена работа с import.meta

## Как проверить себя

1. Запустите демонстрацию и сравните CJS copies vs ESM live bindings
2. Проверьте, что все варианты синтаксиса покрыты
