# Задание 9.3: OS Module

## Цель

Изучить модуль `os` для получения информации о системе: CPU, память, платформа, сетевые интерфейсы.

## Требования

1. Покажите `os.cpus()` — количество ядер, модели, скорость
2. Продемонстрируйте `os.totalmem()` / `os.freemem()` с расчётом использования
3. Покажите информацию о платформе: platform, type, arch, release, hostname
4. Продемонстрируйте `os.networkInterfaces()` — IP-адреса
5. Покажите `os.constants` — сигналы и errno-коды

## Чеклист

- [ ] CPU информация с количеством ядер
- [ ] Память: total, free, used с процентами
- [ ] Все ключевые свойства платформы
- [ ] Сетевые интерфейсы с IPv4/IPv6
- [ ] os.constants для сигналов и errno

## Как проверить себя

1. Нажмите кнопку запуска
2. Убедитесь, что CPU показывает модель и количество ядер
3. Проверьте, что память отображается в человекочитаемых единицах (ГБ)
