# Задание 1.1: CommonJS

## Цель

Разобраться в механизме работы `require()`, `module.exports`, `exports` и кэшировании модулей в CommonJS.

## Требования

1. Продемонстрируйте 5 шагов работы `require()`: Resolve, Cache Check, Load, Wrap, Execute
2. Покажите разницу между `exports` и `module.exports`
3. Симулируйте кэширование модулей: два вызова require возвращают один объект
4. Покажите Module Wrapper Function
5. Объясните порядок поиска файлов при resolve

## Чеклист

- [ ] Показаны все 5 шагов require()
- [ ] Объяснена разница exports vs module.exports с примерами
- [ ] Симуляция кэширования работает корректно
- [ ] Показана wrapper function
- [ ] Компонент использует useState

## Как проверить себя

1. Нажмите "Запустить" и изучите вывод
2. Убедитесь, что кэширование демонстрирует общий объект
3. Проверьте, что объяснена ловушка с переприсвоением exports
