# Задание 2.2: Флаги запуска

## Цель
Изучить основные флаги команды `docker run`.

## Требования
Создайте компонент-справочник флагов с:

1. **Категоризацией** (режим, сеть, окружение, ресурсы и т.д.)
2. Для каждого флага: название, описание, пример использования
3. Покажите минимум 10 флагов

## Ключевые флаги
- `-d`, `-it`, `--rm`, `--name`
- `-p` (порты), `--network`
- `-e`, `--env-file`
- `-v`, `--memory`, `--cpus`, `--restart`

## Проверьте себя
- [ ] Флаги организованы по категориям
- [ ] Есть описание и пример для каждого
- [ ] Показаны минимум 10 флагов
