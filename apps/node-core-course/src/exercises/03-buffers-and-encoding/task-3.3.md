# Задание 3.3: Binary Protocol Parsing

## Цель

Научиться парсить и создавать бинарные данные в формате TLV (Type-Length-Value).

## Требования

1. Реализуйте кодирование сообщения в бинарный формат [type][length][payload]
2. Реализуйте декодирование бинарного сообщения
3. Создайте несколько TLV-записей разных типов (string, uint32, boolean)
4. Покажите парсинг TLV-структуры
5. Перечислите методы Buffer для чтения/записи чисел

## Чеклист

- [ ] Кодирование сообщения в бинарный формат
- [ ] Декодирование из бинарного формата
- [ ] TLV с несколькими типами данных
- [ ] Парсинг TLV работает корректно
- [ ] Методы readUInt/writeUInt перечислены

## Как проверить себя

1. Запустите демонстрацию
2. Проверьте, что закодированное сообщение корректно декодируется
3. Убедитесь, что все TLV-записи правильно парсятся
