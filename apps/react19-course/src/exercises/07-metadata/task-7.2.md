# Задание 7.2: Stylesheet precedence

## Цель

Использовать атрибут `precedence` для управления порядком подключения стилей.

## Требования

1. Создайте компонент с переключением тем
2. Используйте `<link rel="stylesheet" precedence="...">` для подключения стилей
3. Покажите как `precedence` влияет на порядок стилей
4. Продемонстрируйте что React не дублирует стили

## Чеклист

- [ ] `<link>` с атрибутом `precedence` используется в JSX
- [ ] Показана разница между `precedence="default"` и `precedence="high"`
- [ ] Переключение тем работает
- [ ] Код с примером показан

## Как проверить себя

1. Переключите тему — стили меняются
2. Код примера отображает использование precedence
