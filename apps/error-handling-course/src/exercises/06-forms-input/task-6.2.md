# Задание 6.2: Серверные ошибки

## Цель
Обработать ошибки валидации, возвращаемые сервером.

## Требования
1. Создайте форму регистрации с полями username и email
2. Симулируйте серверную валидацию:
   - username="admin" → "Имя уже занято"
   - email с @test.com → "Домен запрещён"
3. При ответе 422 маппируйте ошибки на поля формы
4. Добавьте индикатор загрузки при отправке
5. Покажите успешный результат при валидных данных

## Чеклист
- [ ] Серверная валидация симулирована
- [ ] Ошибки маппируются на поля формы
- [ ] Индикатор загрузки
- [ ] Успешная отправка при валидных данных
