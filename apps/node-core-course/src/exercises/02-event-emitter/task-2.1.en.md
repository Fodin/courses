# Task 2.1: Basic EventEmitter

## Goal

Implement a simplified EventEmitter and demonstrate core API methods.

## Requirements

1. Create a SimpleEventEmitter class with on, off, emit, removeAllListeners methods
2. Demonstrate event subscription and emission
3. Show listener removal (off) and removeAllListeners
4. Display listenerCount and eventNames
5. List the full Node.js EventEmitter API

## Checklist

- [ ] SimpleEventEmitter implemented with on, off, emit
- [ ] Subscription and emission demo works
- [ ] Listener removal is correct
- [ ] listenerCount and eventNames shown
- [ ] Full API listed

## How to verify

1. Run the demo
2. Verify events are correctly added and removed
3. Compare implementation with the full Node.js EventEmitter API
