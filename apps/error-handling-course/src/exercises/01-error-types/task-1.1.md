# Задание 1.1: Встроенные типы ошибок

## Цель

Изучить встроенные типы ошибок JavaScript и научиться их различать через `instanceof`.

## Требования

1. Спровоцируйте каждый тип ошибки в отдельном `try/catch`:
   - `TypeError` — вызов метода на `null`
   - `RangeError` — создание массива с отрицательной длиной
   - `SyntaxError` — парсинг некорректного JSON
   - `URIError` — некорректный `decodeURIComponent`
2. В каждом `catch` проверяйте тип через `instanceof`
3. Покажите цепочку наследования: `TypeError instanceof Error`
4. Отобразите все результаты на странице

- [ ] Результаты отображаются на странице
