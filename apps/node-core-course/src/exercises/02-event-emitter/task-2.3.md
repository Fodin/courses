# Задание 2.3: once() & Async Iterator

## Цель

Научиться использовать одноразовые события (once) и асинхронные итераторы (events.on) для современного async/await кода.

## Требования

1. Продемонстрируйте emitter.once() с автоматическим удалением слушателя
2. Покажите events.once() — Promise-based API
3. Продемонстрируйте events.on() как async iterator
4. Покажите использование AbortSignal для отмены итерации
5. Приведите практические примеры: HTTP server, readline

## Чеклист

- [ ] emitter.once() продемонстрирован с авто-удалением
- [ ] events.once() с Promise показан
- [ ] events.on() с async iterator показан
- [ ] AbortSignal для отмены
- [ ] Практические примеры

## Как проверить себя

1. Запустите демонстрацию
2. Проверьте, что once() слушатель срабатывает только один раз
3. Убедитесь, что Promise и async iterator примеры корректны
