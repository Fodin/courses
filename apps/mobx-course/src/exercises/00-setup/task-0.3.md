# Задание 0.3: Несколько полей

## Цель

Создать `ProfileStore` с несколькими полями и computed-свойством, подключить к React.

## Требования

1. Создайте класс `ProfileStore` со свойствами:
   - `name: string` (по умолчанию `''`)
   - `age: number` (по умолчанию `0`)
   - `bio: string` (по умолчанию `''`)

2. Добавьте методы-сеттеры:
   - `setName(value: string)`
   - `setAge(value: number)`
   - `setBio(value: string)`

3. Добавьте геттер `summary`, возвращающий строку вида `"Имя, N years old"`

4. Создайте экземпляр стора и React-компонент с формой:
   - Input для имени
   - Input (type="number") для возраста
   - Textarea для bio
   - Блок предпросмотра профиля (показывайте `summary` и `bio`)

5. Оберните компонент в `observer`

## Чеклист

- [ ] Класс `ProfileStore` с тремя полями
- [ ] Методы `setName`, `setAge`, `setBio`
- [ ] Геттер `summary` (computed) работает
- [ ] Компонент обёрнут в `observer`
- [ ] Поля ввода обновляют стор
- [ ] Блок предпросмотра отображает актуальные данные

## Как проверить себя

1. Введите имя "Alice" — предпросмотр обновляется
2. Введите возраст 25 — summary показывает "Alice, 25 years old"
3. Введите bio — текст появляется в предпросмотре
