# Задание 5.4: Восстановление загрузки

## Цель
Реализовать загрузку данных с автоматическими повторными попытками и визуализацией прогресса.

## Требования
1. Создайте функцию загрузки, успешную только с 3-й попытки
2. Реализуйте retry-логику с максимум 3 попытками и задержкой 1 секунда
3. Показывайте прогресс: каждая попытка отображается в реальном времени
4. При успехе — показать данные
5. При исчерпании попыток — показать ошибку + кнопку "Заново"

## Чеклист
- [ ] Функция с управляемыми ошибками
- [ ] Retry-логика с 3 попытками
- [ ] Прогресс в реальном времени
- [ ] Успешный результат после retry
- [ ] Кнопка перезапуска при полной неудаче
