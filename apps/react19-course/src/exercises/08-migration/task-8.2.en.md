# Task 8.2: Step-by-Step Migration

## Objective

Create an interactive checklist for a step-by-step migration from React 18 to React 19.

## Requirements

1. Create a list of migration steps (at least 7)
2. Each step contains: title, description, commands (if any), tips
3. Add checkboxes to mark completed steps
4. Show a progress bar
5. Display a congratulation message when all steps are completed

## Checklist

- [ ] Migration steps in the correct order
- [ ] Checkboxes for each step
- [ ] Progress bar updates
- [ ] Terminal commands are displayed in code style
- [ ] Congratulation shown at 100% progress

## How to verify

1. Check off steps — progress grows
2. Check off all steps — a congratulation message appears
