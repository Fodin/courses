# Задание 4.2: Вложенные объекты

## Цель

Создать схему с вложенными объектами и отображать ошибки с путями к конкретным полям.

## Требования

1. Создайте `profileSchema` с вложенными объектами `user` (firstName, lastName) и `address` (city, zipCode, country)

2. `zipCode` должен соответствовать паттерну `/^\d{5,6}$/`

3. Используйте `err.inner` для отображения ошибок с путями (например, `user.firstName: First name is required`)

4. Реализуйте форму с двумя группами полей: User и Address

## Чеклист

- [ ] Схема содержит вложенные объекты `user` и `address`
- [ ] `zipCode` валидируется regex-паттерном
- [ ] Ошибки показывают путь к полю через `err.inner`
- [ ] Формат: `путь: сообщение` для каждой ошибки
- [ ] `abortEarly: false` для сбора всех ошибок

## Как проверить себя

1. Все заполнено — показывает JSON профиля
2. Все пусто — ошибки для каждого вложенного поля с путями
3. Zip "abc" — ошибка `address.zipCode: Invalid zip code`
4. Zip "12345" — успех
