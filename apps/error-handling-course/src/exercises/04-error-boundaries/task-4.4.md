# Задание 4.4: Вложенные Boundaries

## Цель
Построить макет приложения с изолированными секциями через вложенные Error Boundaries.

## Требования
1. Создайте макет с 3 секциями: Header, Content, Sidebar
2. Оберните каждую секцию в отдельный `RecoverableErrorBoundary`
3. Сделайте Sidebar падающим сразу при рендере
4. Сделайте Content падающим по нажатию кнопки
5. Покажите, что ошибка в одной секции не ломает другие

## Чеклист
- [ ] 3 секции в макете
- [ ] Каждая обёрнута в Error Boundary
- [ ] Sidebar падает — Content и Header работают
- [ ] Content падает — Header и Sidebar (ошибка) не затронуты
- [ ] Каждая секция восстанавливается независимо
