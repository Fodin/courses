# Задание 6.4: Доступность ошибок

## Цель
Сделать ошибки формы доступными для screen readers.

## Требования
1. Используйте `aria-required="true"` для обязательных полей
2. Используйте `aria-invalid` при ошибке поля
3. Свяжите ошибку с полем через `aria-describedby`
4. Используйте `role="alert"` и `aria-live` для динамических ошибок
5. При submit фокусируйте первое ошибочное поле
6. В сводке ошибок сделайте ссылки, кликающие на поля

## Чеклист
- [ ] `aria-required` на обязательных полях
- [ ] `aria-invalid` при ошибке
- [ ] `aria-describedby` связывает поле с ошибкой
- [ ] `role="alert"` на контейнере ошибок
- [ ] Фокус на первое ошибочное поле
- [ ] Ссылки в сводке ведут к полям
