# Задание 7.1: exec/execFile

## Цель

Научиться запускать системные команды из Node.js с помощью `exec` и `execFile`, понимать различия между ними и правильно обрабатывать ошибки.

## Требования

1. Создайте симуляцию вызова `exec` с опциями `cwd`, `timeout`, `maxBuffer`
2. Покажите обработку stdout, stderr и error в callback
3. Продемонстрируйте разницу между `exec` и `execFile` (shell injection)
4. Покажите ошибку `maxBuffer exceeded` и способы решения
5. Реализуйте пример с `promisify(exec)` для async/await

## Чеклист

- [ ] Симуляция `exec` с полным набором опций
- [ ] Корректная обработка error, stdout, stderr
- [ ] Демонстрация shell injection уязвимости
- [ ] Показан лимит maxBuffer и решение через spawn
- [ ] Пример Promise-обёртки для exec

## Как проверить себя

1. Нажмите кнопку запуска
2. Убедитесь, что вывод показывает опции exec и их назначение
3. Убедитесь, что демонстрация shell injection наглядна
4. Проверьте, что показаны оба варианта — exec и execFile
