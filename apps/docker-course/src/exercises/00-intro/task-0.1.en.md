# Task 0.1: Containers vs virtual machines

## Objective

Understand the key differences between containers and virtual machines, and learn to determine which approach is better suited for specific scenarios.

## Requirements

1. Create a comparison table of containers and VMs with at least 6 characteristics
2. For each characteristic, specify the value/description for containers and for VMs
3. Use visual indicators (icons/colors) to mark advantages and disadvantages
4. Add a "When to use which" section with specific scenarios
5. Implement interactivity: a button to show/hide a detailed explanation for each characteristic

## Checklist

- [ ] The table contains at least 6 rows with characteristics
- [ ] Values for containers and VMs are provided for each row
- [ ] Advantages are visually highlighted (who wins on each criterion)
- [ ] A use-case scenarios section is present
- [ ] The show/hide details button works correctly
- [ ] The component uses `useState` to manage state

## How to verify

1. Open the task in the browser
2. Make sure the table renders with the correct data
3. Verify that the expand-details button toggles visibility
4. Compare your table with the solution — all key differences should be covered
