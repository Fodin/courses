# Задание 10.2: Symmetric Encryption

## Цель

Освоить симметричное шифрование через `createCipheriv`/`createDecipheriv`, понять разницу между AES-256-GCM и AES-256-CBC, научиться корректно управлять IV.

## Требования

1. Реализуйте симуляцию шифрования и расшифровки текста через AES-256-GCM
2. Покажите генерацию IV (12 байт для GCM) и authTag
3. Продемонстрируйте упаковку iv + authTag + encrypted в единый формат
4. Покажите сравнение GCM и CBC: безопасность, размер IV, аутентификация
5. Продемонстрируйте деривацию ключа из пароля через scrypt

## Чеклист

- [ ] AES-256-GCM шифрование/расшифровка продемонстрированы
- [ ] IV генерируется случайно для каждого шифрования
- [ ] authTag извлечён и использован при расшифровке
- [ ] Показана упаковка данных для хранения/передачи
- [ ] GCM vs CBC сравнение присутствует
- [ ] Деривация ключа через scrypt продемонстрирована

## Как проверить себя

1. Нажмите "Запустить" — должен отобразиться процесс шифрования и расшифровки
2. Убедитесь, что IV уникален для каждого вызова
3. Проверьте наличие authTag при использовании GCM
4. Убедитесь, что объяснена опасность повторного использования IV
