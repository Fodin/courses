# Задание 7.3: Preload API

## Цель

Использовать Preload API React 19 для предзагрузки ресурсов.

## Требования

1. Покажите все функции Preload API: `prefetchDNS`, `preconnect`, `preload`, `preinit`
2. Для каждой функции покажите пример кода и описание
3. Добавьте кнопку «симуляции» вызова каждой функции
4. Объясните когда какую функцию использовать

## Чеклист

- [ ] Все 4+ функции Preload API показаны
- [ ] Каждая функция имеет описание и пример кода
- [ ] Интерактивная симуляция вызова
- [ ] Рекомендации по использованию

## Как проверить себя

1. Все функции отображены с описанием
2. Кнопки «Симулировать» работают
3. Примеры кода читаемы
