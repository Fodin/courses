# Задание 9.1: WebSocket (ws)

## 🎯 Цель

Освоить WebSocket-сервер на библиотеке `ws`: создание сервера, обработка подключений и сообщений, broadcasting и heartbeat.

## Требования

1. Создайте WebSocket-сервер через `WebSocketServer`, привязанный к HTTP-серверу
2. Обработайте события: `connection`, `message` (с JSON-парсингом), `close` (код и причина), `error`
3. Реализуйте broadcast-функцию с возможностью исключить отправителя
4. Покажите heartbeat через ping/pong: обнаружение и terminate "зомби"-соединений
5. Отправьте welcome-сообщение при подключении

## Чеклист

- [ ] WebSocket-сервер создан и слушает подключения
- [ ] Сообщения парсятся из JSON с обработкой ошибок
- [ ] Broadcast отправляет всем клиентам (с проверкой readyState)
- [ ] Heartbeat проверяет соединения каждые 30 секунд
- [ ] Нереспонсивные клиенты удаляются через terminate

## Как проверить себя

Нажмите "Запустить" и убедитесь, что: сервер принимает подключения, сообщения обрабатываются и транслируются, heartbeat обнаруживает отключившихся клиентов.
