# Задание 6.1: Серверные vs клиентские

## Цель

Научиться определять, какой компонент должен быть серверным, а какой — клиентским.

## Требования

1. Создайте интерактивную карточную игру
2. Покажите карточки с описанием компонентов
3. Пользователь выбирает: «Серверный» или «Клиентский»
4. После ответа покажите правильный ответ с объяснением

## Чеклист

- [ ] Минимум 5 карточек с компонентами
- [ ] Кнопки выбора «Серверный» / «Клиентский»
- [ ] Правильный/неправильный ответ подсвечивается
- [ ] Объяснение почему компонент серверный/клиентский
- [ ] Счётчик правильных ответов

## Как проверить себя

1. Прочитайте описание компонента
2. Выберите тип — серверный или клиентский
3. Проверьте объяснение
