# Задание 1.3: ref cleanup function

## Цель

Создать компонент с ref callback, который возвращает cleanup-функцию (новая возможность React 19).

## Требования

1. Создайте компонент с кнопкой для показа/скрытия `<input>`
2. На `<input>` установите ref callback:
   ```tsx
   ref={(node) => {
     // Setup: элемент появился в DOM
     node.focus()

     // Cleanup: элемент удалён из DOM
     return () => {
       console.log('Элемент удалён!')
     }
   }}
   ```
3. При показе input — автоматически устанавливайте на него фокус (в setup)
4. При скрытии input — логируйте событие (в cleanup)
5. Добавьте визуальный лог событий на странице

## Паттерн

```tsx
// React 18 (старый паттерн)
ref={(node) => {
  if (node) { /* setup */ }
  else { /* cleanup (node === null) */ }
}}

// React 19 (новый паттерн)
ref={(node) => {
  /* setup */
  return () => { /* cleanup */ }
}}
```

## Чеклист

- [ ] ref callback возвращает cleanup-функцию
- [ ] При монтировании input получает фокус
- [ ] При размонтировании выполняется cleanup
- [ ] Лог событий отображается на странице
