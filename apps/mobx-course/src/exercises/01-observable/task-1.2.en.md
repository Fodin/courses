# Task 1.2: makeObservable

## Goal
Create a store with **explicit** annotations using `makeObservable`.

## Requirements

- [ ] Create `UserStore` with `firstName` and `lastName` fields
- [ ] Use `makeObservable(this, {...})` with explicit annotations
- [ ] Mark fields as `observable`, methods as `action`, getters as `computed`
- [ ] Add `fullName` getter — concatenation of first and last name
- [ ] Add `initials` getter — first letters of names
- [ ] Display all data in an observer component
