# Задание 5.1: array().of()

## Цель

Научиться создавать массивные схемы с валидацией каждого элемента через `array().of()`.

## Требования

1. Создайте `tagsSchema` — массив строк, каждая строка обязательна (`string().required()`)

2. Создайте `numbersSchema` — массив чисел, каждое число обязательно и положительно

3. Парсите ввод из текстового поля по запятой: `input.split(',').map(s => s.trim())`

4. При успехе покажите валидированный массив в зелёном блоке

5. При ошибке покажите сообщение в красном блоке

## Чеклист

- [ ] `tagsSchema` использует `array().of(string().required())`
- [ ] `numbersSchema` использует `array().of(number().required().positive())`
- [ ] Ввод парсится из строки через split по запятой
- [ ] Пустой ввод передаётся как undefined
- [ ] Для numbers используется `abortEarly: false`

## Как проверить себя

1. Tags: "react, typescript, yup" — успех, показывает массив
2. Tags: пусто — ошибка required
3. Numbers: "1, 2, 3" — успех
4. Numbers: "1, -5, 3" — ошибка "Each number must be positive"
5. Numbers: "1, abc, 3" — ошибка (NaN не число)
