# Task 7.2: Stylesheet Precedence

## Objective

Use the `precedence` attribute to control the order in which stylesheets are applied.

## Requirements

1. Create a component with theme switching
2. Use `<link rel="stylesheet" precedence="...">` to apply styles
3. Show how `precedence` affects the style order
4. Demonstrate that React does not duplicate styles

## Checklist

- [ ] `<link>` with the `precedence` attribute is used in JSX
- [ ] The difference between `precedence="default"` and `precedence="high"` is shown
- [ ] Theme switching works
- [ ] A code example is displayed

## How to verify

1. Switch the theme — styles change
2. The code example shows the use of precedence
