# Задание 7.3: Отображение ошибок

## Цель
Реализовать три паттерна отображения ошибок: inline, сводка, toast.

## Требования
1. **Inline ошибка** — под конкретным полем
2. **Сводка ошибок** — блок над формой со списком всех ошибок
3. **Toast-уведомление** — временное сообщение, исчезающее через 3 секунды
4. Сводка должна использовать `role="alert"`
5. Покажите все 3 паттерна по нажатию кнопок

## Чеклист
- [ ] Inline ошибка под полем
- [ ] Сводка ошибок над формой
- [ ] Toast-уведомление с автоскрытием
- [ ] `role="alert"` на сводке
