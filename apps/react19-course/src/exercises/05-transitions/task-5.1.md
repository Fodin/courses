# Задание 5.1: useTransition + isPending

## Цель

Использовать `useTransition` для переключения вкладок без блокировки UI.

## Требования

1. Создайте компонент с 3 вкладками, каждая «загружает» данные
2. Используйте `useTransition` для переключения
3. Покажите `isPending` как индикатор загрузки (opacity или spinner)
4. Текущая вкладка остаётся видимой, пока новая загружается

## Чеклист

- [ ] `useTransition` возвращает `[isPending, startTransition]`
- [ ] Переключение вкладок обёрнуто в `startTransition`
- [ ] `isPending` показывает индикатор загрузки
- [ ] UI не блокируется при переключении

## Как проверить себя

1. Нажмите на вкладку — текущая контент слегка затемняется
2. Через секунду новый контент появляется
