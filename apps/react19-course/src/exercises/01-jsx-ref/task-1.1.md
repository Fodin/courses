# Задание 1.1: JSX без import React

## Цель

Убрать ненужный `import React from 'react'` из компонента, используя новый JSX transform.

## Требования

1. Откройте файл задания — в нём есть `import React from 'react'`
2. Убедитесь, что компонент использует JSX (возвращает `<div>`)
3. Удалите строку `import React from 'react'`
4. Убедитесь, что компонент по-прежнему работает
5. Если нужен `useState` или другой хук — импортируйте его напрямую:
   ```tsx
   import { useState } from 'react'
   ```

## Объяснение

Новый JSX transform (введён в React 17, обязателен в React 19) автоматически добавляет импорт из `react/jsx-runtime`. Вам больше не нужно импортировать `React` для использования JSX.

## Чеклист

- [ ] Удалён `import React from 'react'`
- [ ] Компонент работает без ошибок
- [ ] Если используются хуки — они импортированы напрямую
- [ ] В `tsconfig.json` установлен `"jsx": "react-jsx"`
