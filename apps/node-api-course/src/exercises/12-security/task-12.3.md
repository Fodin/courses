# Задание 12.3: CSRF & Advanced Security

## 🎯 Цель

Освоить продвинутые техники безопасности: CSRF-токены, HTTP parameter pollution, ReDoS prevention.

## Требования

1. Реализуйте CSRF-защиту: csurf middleware, endpoint для получения токена, проверка в POST-запросах
2. Покажите double-submit cookie паттерн для SPA
3. Подключите hpp (HTTP Parameter Pollution) с whitelist
4. Покажите ReDoS-уязвимость и защиту через RE2
5. Продемонстрируйте SameSite cookies как современную альтернативу CSRF-токенам

## Чеклист

- [ ] CSRF-токен генерируется и проверяется
- [ ] POST без CSRF-токена возвращает 403
- [ ] HPP дедуплицирует параметры запроса
- [ ] ReDoS-уязвимая regex показана и заменена на RE2
- [ ] SameSite cookies настроены как дополнительная защита

## Как проверить себя

Нажмите "Запустить" и убедитесь, что: CSRF-токен работает, запросы без токена блокируются, HPP защищает от дублирования параметров, RE2 работает в линейном времени.
