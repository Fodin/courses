# Задание 5.3: Алгебраические типы данных

## Цель

Научиться моделировать сложные домены с помощью алгебраических типов данных (ADT): Option, Result и рекурсивные деревья.

## Требования

1. Реализуйте тип `Option<T>` (варианты `some` и `none`) с функциями-конструкторами `some()` и `none()`
2. Создайте функции `mapOption`, `unwrapOr` для работы с `Option`
3. Реализуйте тип `Result<T, E>` (варианты `ok` и `err`) с функциями-конструкторами
4. Создайте функцию `parseAge(input: string): Result<number, string>`, валидирующую входные данные
5. Реализуйте рекурсивный ADT `Expr` для дерева арифметических выражений с операциями `num`, `add`, `mul`, `neg`
6. Создайте функции `evaluate(expr)` и `printExpr(expr)` для вычисления и отображения выражений

## Чеклист

- [ ] `Option<T>` реализован с `some` и `none` вариантами
- [ ] `mapOption` трансформирует значение внутри `some`, пропускает `none`
- [ ] `unwrapOr` возвращает значение или дефолт
- [ ] `Result<T, E>` реализован с `ok` и `err` вариантами
- [ ] `parseAge` корректно валидирует и возвращает `Result`
- [ ] `Expr` поддерживает рекурсивные выражения
- [ ] `evaluate` вычисляет дерево выражений
- [ ] `printExpr` строит строковое представление

## Как проверить себя

1. Нажмите кнопку «Запустить»
2. Убедитесь, что `mapOption(some(user), u => u.name)` возвращает `some` с именем
3. Убедитесь, что `parseAge("abc")` возвращает `err` с сообщением
4. Убедитесь, что выражение `(2 + 3) * (-4)` вычисляется как `-20`
