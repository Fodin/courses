# Задание 2.3: Bound actions

## Цель
Использовать `autoBind: true` чтобы методы не теряли `this` при передаче как callback.

## Требования

- [ ] Создайте класс `TimerStore` с полями `seconds` и `running`
- [ ] Используйте `makeAutoObservable(this, {}, { autoBind: true })`
- [ ] Добавьте метод `start()` — запускает `setInterval` с `this.tick`
- [ ] Добавьте метод `tick()` — увеличивает `seconds` на 1
- [ ] Добавьте метод `stop()` — останавливает интервал
- [ ] Добавьте метод `reset()` — останавливает и сбрасывает секунды
- [ ] Добавьте геттер `formatted` — формат `MM:SS`
- [ ] Передайте методы напрямую в `onClick` (без стрелочной обёртки)
