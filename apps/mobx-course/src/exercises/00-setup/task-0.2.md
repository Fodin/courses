# Задание 0.2: Стор + React

## Цель

Подключить `CounterStore` к React-компоненту через `observer` из `mobx-react-lite`.

## Требования

1. Импортируйте `observer` из `'mobx-react-lite'`

2. Создайте класс `CounterStore` с `makeAutoObservable` (как в задании 0.1)

3. Создайте экземпляр стора **вне компонента** (на уровне модуля)

4. Оберните компонент в `observer`:
   ```tsx
   export const Task0_2 = observer(function Task0_2() {
     // ...
   })
   ```

5. Отобразите текущее значение `count` и кнопки "+1" / "-1"

## Чеклист

- [ ] `observer` импортирован из `'mobx-react-lite'`
- [ ] Компонент обёрнут в `observer`
- [ ] Стор создан вне компонента
- [ ] Кнопка "+1" вызывает `increment()`
- [ ] Кнопка "-1" вызывает `decrement()`
- [ ] Значение `count` обновляется на экране при нажатии кнопок

## Как проверить себя

1. Нажмите "+1" три раза — счётчик должен показать `3`
2. Нажмите "-1" один раз — счётчик должен показать `2`
3. Убедитесь, что число на экране обновляется мгновенно
