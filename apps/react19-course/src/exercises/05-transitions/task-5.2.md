# Задание 5.2: Async transitions

## Цель

Использовать async-функцию в `startTransition` — новая возможность React 19.

## Требования

1. Создайте `startTransition` через `useTransition`
2. Передайте в `startTransition` async-функцию с await
3. `isPending` автоматически станет true на время await
4. Покажите сохранение данных с индикатором

## Чеклист

- [ ] Async-функция передана в `startTransition`
- [ ] `await` используется внутри transition
- [ ] `isPending` автоматически отслеживает async-операцию
- [ ] Индикатор загрузки отображается во время await

## Как проверить себя

1. Нажмите «Сохранить» — появится индикатор загрузки
2. После завершения async-операции индикатор исчезнет
