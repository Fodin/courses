# Задание 2.1: docker run

## Цель
Понять, как запускать контейнеры и что происходит при вызове `docker run`.

## Требования
Создайте компонент, отображающий:

1. **Несколько примеров** `docker run` с разными образами и параметрами
2. Для каждого примера — **пошаговое описание** того, что происходит внутри Docker
3. Объясните формат: `docker run [ОПЦИИ] ОБРАЗ [КОМАНДА] [АРГУМЕНТЫ]`

## Подсказки
- `docker run ubuntu echo "Hello"` — foreground, контейнер сразу завершится
- `docker run -it ubuntu bash` — интерактивная оболочка
- `docker run -d nginx` — фоновый режим
- `docker run --rm alpine echo hi` — автоудаление после завершения
