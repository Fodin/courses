# Task 2.3: once() & Async Iterator

## Goal

Learn to use one-time events (once) and async iterators (events.on) for modern async/await code.

## Requirements

1. Demonstrate emitter.once() with automatic listener removal
2. Show events.once() — Promise-based API
3. Demonstrate events.on() as async iterator
4. Show AbortSignal usage for cancellation
5. Provide practical examples: HTTP server, readline
