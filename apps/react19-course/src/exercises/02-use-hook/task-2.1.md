# Задание 2.1: use(Promise)

## Цель

Использовать новый API `use()` для чтения данных из промиса с отображением загрузки через Suspense.

## Требования

1. Создайте промис **вне компонента**, который возвращает массив пользователей через `setTimeout`
2. Создайте дочерний компонент `UserList`, который вызывает `use(promise)` для получения данных
3. Отобразите список пользователей (имя и роль)
4. Оберните `UserList` в `<Suspense>` с fallback-сообщением о загрузке

## Чеклист

- [ ] Промис создан вне компонента
- [ ] `use()` импортирован из `'react'`
- [ ] `use(promise)` вызывается в дочернем компоненте
- [ ] `<Suspense>` оборачивает компонент с `use()`
- [ ] Во время загрузки отображается fallback
- [ ] После загрузки отображается список пользователей

## Как проверить себя

1. При первой загрузке видно сообщение «Загрузка...»
2. Через 1-2 секунды появляется список пользователей
