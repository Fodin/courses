# Задание 2.1: Basic Mapped Types

## Цель

Научиться создавать базовые mapped types: свои версии Readonly, Partial, Required, Pick, Record и кастомный Nullable.

## Требования

1. Реализуйте `MyReadonly<T>` -- делает все свойства readonly. Покажите, что попытка мутации даёт ошибку
2. Реализуйте `MyPartial<T>` -- делает все свойства опциональными
3. Реализуйте `MyRequired<T>` с `-?` -- убирает опциональность всех свойств
4. Реализуйте `MyPick<T, K extends keyof T>` -- выбирает подмножество ключей из типа
5. Реализуйте `MyRecord<K extends string | number | symbol, V>` -- создаёт тип с заданными ключами и типом значений
6. Реализуйте `Nullable<T>` -- делает все значения допускающими null

## Чеклист

- [ ] `MyReadonly<User>` не допускает присваивание свойствам
- [ ] `MyPartial<User>` позволяет создать объект с неполным набором свойств
- [ ] `MyRequired<Config>` требует указания всех свойств, включая ранее опциональные
- [ ] `MyPick<User, 'name' | 'email'>` содержит только указанные ключи
- [ ] `MyRecord<'a' | 'b', number>` содержит ключи 'a' и 'b' с числовыми значениями
- [ ] `Nullable<User>` допускает null для каждого свойства

## Как проверить себя

1. Попробуйте присвоить значение readonly-свойству -- должна быть ошибка
2. Создайте объект MyPartial без некоторых свойств -- должно компилироваться
3. Создайте объект MyRequired без свойства -- должна быть ошибка
