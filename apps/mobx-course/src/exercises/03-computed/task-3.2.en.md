# Task 3.2: Computed Chain

## Goal
Build a chain of computed values: subtotal -> tax -> total.

## Requirements

- [ ] Create `InvoiceStore` with items array and `taxRate`
- [ ] Add computed `subtotal` — sum without tax
- [ ] Add computed `tax` — `subtotal * taxRate`
- [ ] Add computed `total` — `subtotal + tax`
- [ ] Allow changing the tax rate
- [ ] Display all three values in component
