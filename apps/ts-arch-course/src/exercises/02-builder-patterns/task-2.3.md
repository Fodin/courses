# Задание 2.3: Условные методы билдера

## 🎯 Цель

Реализовать Builder для конфигурации базы данных, в котором доступные методы зависят от выбранного типа БД.

## Требования

1. Поддержите минимум 3 типа БД: PostgreSQL, MySQL, SQLite
2. Общие методы (database, logging) доступны для всех типов
3. Сетевые методы (host, port) доступны только для PostgreSQL и MySQL
4. Специфичные методы:
   - PostgreSQL: ssl, poolSize
   - MySQL: charset
   - SQLite: filename, mode
5. Используйте this-parameter typing для ограничения доступных методов
6. build() доступен только после выбора типа БД

## Чеклист

- [ ] `builder.sqlite().host("x")` — ошибка компиляции
- [ ] `builder.mysql().ssl(true)` — ошибка (ssl только для postgres)
- [ ] `builder.postgres().filename("x")` — ошибка (filename только для sqlite)
- [ ] `new DbConfigBuilder().build()` — ошибка (не выбран тип БД)
- [ ] Каждый тип БД имеет свои специфичные настройки по умолчанию

## Как проверить себя

Создайте конфигурации для всех трёх типов БД. Попробуйте вызвать специфичные методы на неподходящем типе — TypeScript должен показать ошибку. Проверьте, что дефолтные значения (порт, charset) корректны.
