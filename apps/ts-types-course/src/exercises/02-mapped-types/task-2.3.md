# Задание 2.3: Modifier Manipulation

## Цель

Научиться управлять модификаторами `readonly` и `optional` (`?`) в mapped types: добавлять, удалять и комбинировать их, включая применение к подмножеству ключей.

## Требования

1. Реализуйте `Mutable<T>` -- удаляет `readonly` со всех свойств через `-readonly`
2. Реализуйте `Concrete<T>` -- удаляет `?` со всех свойств через `-?`
3. Реализуйте `ReadonlyRequired<T>` -- одновременно добавляет `readonly` и убирает `?`
4. Реализуйте `ReadonlyPick<T, K>` -- делает только указанные ключи readonly, остальные оставляет мутабельными
5. Реализуйте `OptionalExcept<T, K>` -- делает все свойства опциональными, КРОМЕ указанных ключей

## Чеклист

- [ ] `Mutable<{readonly a: string}>` позволяет мутировать `a`
- [ ] `Concrete<{a?: string}>` требует указания `a`
- [ ] `ReadonlyRequired` комбинирует оба модификатора
- [ ] `ReadonlyPick<Doc, 'id'>` -- `id` readonly, остальные мутабельные
- [ ] `OptionalExcept<Form, 'username'>` -- `username` обязательный, остальные опциональные

## Как проверить себя

1. Попробуйте мутировать свойство объекта `Mutable<FrozenType>` -- должно работать
2. Создайте объект `Concrete<PartialType>` без свойства -- ошибка
3. Проверьте `OptionalExcept` -- создайте объект только с обязательными ключами
