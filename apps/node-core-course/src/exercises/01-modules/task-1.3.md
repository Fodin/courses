# Задание 1.3: Circular Dependencies

## Цель

Понять, как Node.js обрабатывает циклические зависимости, и научиться их избегать.

## Требования

1. Покажите пример циклической зависимости в CommonJS
2. Симулируйте пошаговое выполнение с partial exports
3. Объясните, почему импортированный модуль может быть "частичным"
4. Продемонстрируйте стратегии решения: рефакторинг, lazy require, dependency injection
5. Объясните отличия обработки циклов в ESM

## Чеклист

- [ ] Показан пример цикла A → B → A
- [ ] Пошаговая симуляция с partial exports
- [ ] Объяснена причина частичной загрузки
- [ ] Показаны минимум 3 стратегии решения
- [ ] Упомянуты отличия ESM

## Как проверить себя

1. Запустите симуляцию и проследите порядок загрузки
2. Убедитесь, что видно, когда модуль возвращает partial exports
