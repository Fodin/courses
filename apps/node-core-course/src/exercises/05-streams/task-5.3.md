# Задание 5.3: Transform Streams

## Цель

Научиться создавать Transform Streams для преобразования данных на лету, включая objectMode и цепочки трансформаций.

## Требования

1. Покажите создание Transform Stream с методом `transform()`
2. Продемонстрируйте метод `flush()` для финализации
3. Покажите `objectMode` для работы с JS-объектами вместо буферов
4. Продемонстрируйте цепочку трансформаций (pipe chain)
5. Симулируйте цепочку: Trim → UpperCase → AddPrefix
6. Симулируйте objectMode: Parse JSON → Filter → Output

## Чеклист

- [ ] Создание Transform через new Transform() показано
- [ ] flush() для финализации продемонстрирован
- [ ] objectMode объяснён и показан в примере
- [ ] Цепочка трансформаций работает
- [ ] Симуляция string-трансформаций корректна
- [ ] Симуляция objectMode с фильтрацией работает

## Как проверить себя

1. Нажмите "Запустить" и проследите цепочку трансформаций
2. Убедитесь, что каждый шаг трансформации показан отдельно
3. Проверьте, что objectMode фильтрует объекты по условию
4. Убедитесь, что callback() вызывается в каждом transform
