# Задание 0.2: React Codemod

## Цель

Создать компонент, демонстрирующий типичные паттерны кода, которые codemod автоматически заменяет при миграции на React 19.

## Требования

1. Создайте массив примеров codemod-преобразований (минимум 4)
2. Для каждого преобразования покажите:
   - Название трансформации
   - Код **до** (React 18)
   - Код **после** (React 19)
3. Добавьте навигацию между примерами (кнопки или табы)
4. Покажите команду запуска codemod

## Примеры преобразований

| До (React 18) | После (React 19) |
|----------------|-------------------|
| `ReactDOM.render(<App />, el)` | `createRoot(el).render(<App />)` |
| String ref: `ref="myInput"` | `createRef()` / `useRef` |
| `Button.defaultProps = { color: 'blue' }` | `function Button({ color = 'blue' })` |
| `forwardRef((props, ref) => ...)` | `function Comp({ ref, ...props })` |

## Подсказки

- Используйте `useState` для отслеживания активного примера
- Код примеров храните как строки в массиве объектов
- Используйте `<pre>` для отображения кода
