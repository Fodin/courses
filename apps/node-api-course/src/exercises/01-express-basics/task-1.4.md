# Задание 1.4: Шаблонизаторы (EJS)

## 🎯 Цель

Настроить шаблонизатор EJS в Express: рендеринг views, передача данных, использование partials и структура директории views.

## Требования

1. Настройте EJS: `app.set('view engine', 'ejs')` и `app.set('views', ...)`
2. Покажите структуру директории views: layouts, partials, pages
3. Продемонстрируйте синтаксис EJS: `<%= %>` (экранированный вывод), `<%- %>` (raw), `<% %>` (код), `<%- include() %>`
4. Реализуйте рендеринг страницы со списком пользователей через `res.render('template', data)`
5. Покажите результат рендеринга в виде HTML

## Чеклист

- [ ] EJS настроен через `app.set('view engine', 'ejs')`
- [ ] Структура views организована: layouts/, partials/, pages/
- [ ] Показан синтаксис: экранированный вывод, raw, управляющие конструкции, include
- [ ] `res.render()` вызывается с именем шаблона и объектом данных
- [ ] Результат рендеринга содержит данные из контекста и подключённые partials

## Как проверить себя

Нажмите "Запустить" и убедитесь, что: настройка EJS показана корректно, синтаксис описан, рендеринг формирует HTML с данными из контекста и включёнными partials.
