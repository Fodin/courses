# Задание 2.1: Promise rejection

## Цель
Научиться обрабатывать отклонённые промисы через `.catch()`.

## Требования
1. Создайте промис, который отклоняется через `setTimeout` с `Error('Таймаут')`
2. Обработайте через `.then().catch()`
3. Покажите работу `Promise.reject`
4. Продемонстрируйте, как ошибка «проваливается» через цепочку `.then` до `.catch`
5. Покажите, что `.then` после `.catch` выполняется

## Чеклист
- [ ] Промис с таймаутом создан и обработан
- [ ] `Promise.reject` показан
- [ ] Цепочка промисов с ошибкой в середине
- [ ] Код после `.catch` выполняется
