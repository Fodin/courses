# Задание 9.2: Тестирование ошибок

## Цель
Научиться писать тесты, проверяющие корректность обработки ошибок.

## Требования
1. Создайте мини-фреймворк: функции `test(name, fn)` и `expect(value)`
2. Реализуйте методы: `toBe(expected)`, `toThrow(fn, message?)`, `toThrowType(fn, ErrorType)`
3. Напишите тесты для:
   - `divideOrThrow` — обычное деление и деление на ноль
   - `validateAge` — валидный возраст, строка (TypeError), отрицательное (RangeError)
4. Отобразите результаты: зелёный/красный для каждого теста

## Чеклист
- [ ] `test` и `expect` реализованы
- [ ] `toBe`, `toThrow`, `toThrowType` работают
- [ ] Минимум 6 тестов
- [ ] Визуальный отчёт: passed/failed
