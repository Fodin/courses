# Задание 11.3: Чеклист безопасности

## Цель

Собрать комплексный интерактивный чеклист безопасности Docker, покрывающий read-only filesystem, ограничение ресурсов, сетевую изоляцию и управление секретами. Создать компонент с отслеживанием прогресса.

## Требования

1. Создайте компонент с интерактивным чеклистом безопасности, разбитым на 5 категорий: "Образы", "Dockerfile", "Runtime", "Секреты", "Сеть"
2. Каждая категория содержит 4-6 пунктов чеклиста с чекбоксами, которые можно отмечать
3. Для каждого пункта показывайте: текст рекомендации, блок кода с примером команды/конфигурации (по клику "Показать пример")
4. Вверху компонента -- прогресс-бар, показывающий процент выполнения всего чеклиста (сколько пунктов отмечено из общего числа)
5. Под прогресс-баром -- отдельные мини-прогресс-бары для каждой категории
6. Добавьте кнопку "Показать безопасный docker-compose.yml" -- по клику отображается полный пример production-ready docker-compose.yml со всеми мерами безопасности
7. Внизу компонента -- блок "Defense in Depth" с визуализацией слоёв защиты

## Подсказки

- `useState<Record<string, boolean>>` для состояния чекбоксов (ключ = уникальный ID пункта)
- `useState<Record<string, boolean>>` для видимости примеров кода
- `useState<boolean>` для показа полного docker-compose.yml
- Прогресс: `Object.values(checked).filter(Boolean).length / totalItems * 100`
- Группировка пунктов по категориям: массив объектов `{ category, items: [{ id, text, code }] }`

## Чеклист

- [ ] 5 категорий чеклиста
- [ ] Минимум 22 пункта с чекбоксами суммарно
- [ ] Кнопки "Показать пример" для каждого пункта
- [ ] Общий прогресс-бар (процент выполнения)
- [ ] Мини-прогресс-бары для каждой категории
- [ ] Кнопка "Показать безопасный docker-compose.yml"
- [ ] Блок "Defense in Depth" с визуализацией слоёв

## Как проверить себя

1. Отмечайте чекбоксы -- прогресс-бар обновляется
2. Мини-прогресс-бары обновляются при отметке пунктов соответствующей категории
3. Кнопки "Показать пример" работают независимо для каждого пункта
4. Кнопка docker-compose.yml показывает полный пример
5. Блок "Defense in Depth" визуализирует слои защиты
6. При 100% прогрессе виден индикатор завершения
