# Задание 2.4: Батчинг

## Цель
Убедиться, что несколько мутаций в одном action вызывают только один ререндер.

## Требования

- [ ] Создайте класс `ProfileStore` с полями `firstName`, `lastName`, `age`
- [ ] Используйте `makeAutoObservable(this)` в конструкторе
- [ ] Добавьте метод `updateProfile(first, last, age)` — обновляет все 3 поля
- [ ] Добавьте геттер `fullName`
- [ ] В компоненте используйте `useRef` для подсчёта ререндеров
- [ ] Оберните компонент в `observer`
- [ ] Убедитесь, что при нажатии кнопки счётчик ререндеров увеличивается на 1 (не на 3)
