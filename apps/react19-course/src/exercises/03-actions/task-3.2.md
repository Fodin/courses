# Задание 3.2: useActionState

## Цель

Управлять состоянием формы через `useActionState` — новый хук React 19, объединяющий action и состояние.

## Требования

1. Импортируйте `useActionState` из `'react'`
2. Создайте action-функцию `(prevState, formData) => newState`
3. Используйте `useActionState(action, initialState)`
4. Покажите ошибки валидации и статус успеха
5. Добавьте имитацию задержки сервера

## Чеклист

- [ ] `useActionState` возвращает `[state, formAction, isPending]`
- [ ] Action-функция обрабатывает FormData и возвращает новое состояние
- [ ] Ошибки валидации отображаются
- [ ] isPending используется для показа загрузки
- [ ] Успешная отправка показывает результат

## Как проверить себя

1. Отправьте пустую форму — появятся ошибки валидации
2. Заполните все поля и отправьте — появится индикатор загрузки, затем результат
