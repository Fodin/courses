# Задание 0.1: Первый стор

## Цель

Создать класс `CounterStore` с использованием `makeAutoObservable` и проверить его работу через консоль (без React).

## Требования

1. Импортируйте `makeAutoObservable` из `'mobx'`

2. Создайте класс `CounterStore` со свойством `count = 0`

3. В конструкторе вызовите `makeAutoObservable(this)`

4. Добавьте методы:
   - `increment()` — увеличивает `count` на 1
   - `decrement()` — уменьшает `count` на 1

5. Создайте экземпляр стора и выполните операции:
   - Три раза `increment`
   - Один раз `decrement`
   - Выведите `store.count` в консоль (ожидается `2`)

## Чеклист

- [ ] `makeAutoObservable` импортирован из `'mobx'`
- [ ] Класс `CounterStore` определён
- [ ] Свойство `count` инициализировано значением `0`
- [ ] `makeAutoObservable(this)` вызван в конструкторе
- [ ] Методы `increment` и `decrement` работают
- [ ] В консоли выводится `2`

## Как проверить себя

1. Откройте DevTools (F12) -> Console
2. Убедитесь, что в консоли видно `count: 2`
3. Компонент отображает текст с результатом
