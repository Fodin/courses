# Задание 7.2: spawn

## Цель

Освоить потоковый запуск процессов через `spawn`, управление stdio, обработку сигналов и создание detached-процессов.

## Требования

1. Создайте симуляцию `spawn` с потоковым выводом stdout/stderr
2. Покажите разные режимы stdio: pipe, inherit, ignore
3. Продемонстрируйте отправку сигналов: SIGTERM, SIGKILL, SIGINT
4. Реализуйте пример detached-процесса с `unref()`
5. Покажите обработку событий: data, close, error

## Чеклист

- [ ] Потоковый вывод stdout и stderr с цветовой маркировкой
- [ ] Объяснение режимов stdio
- [ ] Демонстрация сигналов и их различий
- [ ] Пример detached + unref
- [ ] Обработка события error (ENOENT)

## Как проверить себя

1. Нажмите кнопку запуска
2. Убедитесь, что stdout (зелёный) и stderr (красный) различаются визуально
3. Проверьте, что показаны все режимы stdio
4. Убедитесь, что объяснены различия сигналов
