# Task 2.4: docker exec

## Objective
Learn to run commands inside a running container.

## Requirements
Create a component with `docker exec` examples:

1. Show **typical debugging scenarios** (files, network, environment, database)
2. Explain the key flags: `-it`, `-u`, `-w`, `-e`
3. Add examples for different situations

## Checklist
- [ ] The command syntax is shown
- [ ] At least 8 examples with descriptions
- [ ] Key flags are explained
