# Задание 8.2: Сервис логирования

## Цель
Создать класс ErrorLogger для централизованного логирования ошибок.

## Требования
1. Создайте класс `ErrorLogger` с методами `error()`, `warn()`, `info()`
2. Каждый лог содержит: level, message, timestamp, context?, stack?
3. Реализуйте подписку через `subscribe(fn)` для React-интеграции
4. Добавьте метод `clear()` для очистки
5. Отобразите логи в виде таблицы с цветовой индикацией по уровню

## Чеклист
- [ ] Класс `ErrorLogger` с 3 методами
- [ ] Подписка через subscribe/unsubscribe
- [ ] Логи отображаются в таблице
- [ ] Цветовая индикация по уровню
- [ ] Очистка логов
