# Задание 2.3: Условный use()

## Цель

Продемонстрировать уникальную возможность `use()` — вызов внутри условий, что невозможно с обычными хуками.

## Требования

1. Создайте `UserContext` с данными пользователя
2. Создайте компонент `Greeting`, принимающий prop `showUser: boolean`
3. Внутри `Greeting` вызовите `use(UserContext)` **только если** `showUser === true`
4. Если `showUser === false`, покажите сообщение «Контекст не используется»
5. Добавьте кнопку для toggle `showUser`

## Чеклист

- [ ] `use(UserContext)` вызывается внутри `if`
- [ ] При `showUser = true` отображаются данные пользователя
- [ ] При `showUser = false` отображается альтернативное сообщение
- [ ] Кнопка переключает состояние
- [ ] Нет ошибок при переключении

## Как проверить себя

1. По умолчанию показывается «Контекст не используется»
2. После нажатия кнопки появляются данные пользователя из контекста
