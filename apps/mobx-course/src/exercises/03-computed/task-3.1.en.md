# Task 3.1: First Computed

## Goal
Create a cart store with a computed total price.

## Requirements

- [ ] Create `CartStore` with `items: {name, price, qty}[]` array
- [ ] Add computed getter `totalPrice` — sum of `price * qty`
- [ ] Add methods `setQty`, `addItem`, `removeItem`
- [ ] Display item table with +/- buttons for quantity
- [ ] Show total price (updates automatically)
