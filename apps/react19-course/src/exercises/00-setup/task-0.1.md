# Задание 0.1: Обновление до React 19

## Цель

Создать компонент, который наглядно показывает процесс обновления с React 18 на React 19.

## Требования

1. Создайте две секции: **"До обновления"** и **"После обновления"**
2. В каждой секции покажите содержимое `package.json` (зависимости)
3. Покажите какие пакеты нужно обновить:
   - `react` — 18.x → 19.x
   - `react-dom` — 18.x → 19.x
   - `@types/react` — 18.x → 19.x
   - `@types/react-dom` — 18.x → 19.x
4. Добавьте кнопку для показа команд обновления (`npm install ...`)
5. Перечислите дополнительные пакеты, которые нужно обновить (eslint-plugin, testing-library и т.д.)

## Подсказки

- Используйте `useState` для toggle показа команд
- Данные версий можно хранить как объекты
- Используйте `JSON.stringify(obj, null, 2)` для красивого вывода JSON
