# Задание 4.4: Infer в кортежах

## Цель

Научиться использовать `infer` для манипуляций с кортежами (tuple types): извлечение элементов, разворот, слияние и трансформация.

## Требования

1. Создайте типы `Head<T>` и `Last<T>` для извлечения первого и последнего элементов кортежа
2. Создайте тип `Tail<T>`, возвращающий кортеж без первого элемента
3. Создайте тип `Reverse<T>`, рекурсивно разворачивающий кортеж (`[1, 2, 3]` -> `[3, 2, 1]`)
4. Создайте тип `FlattenOnce<T>`, который раскрывает один уровень вложенности массивов в кортеже (`[[1, 2], [3]]` -> `[1, 2, 3]`)
5. Создайте тип `Zip<A, B>`, объединяющий два кортежа попарно (`['a', 'b'], [1, 2]` -> `[['a', 1], ['b', 2]]`)
6. Продемонстрируйте вычисление длины кортежа через `T['length']`

## Чеклист

- [ ] `Head` и `Last` корректно извлекают элементы из кортежа
- [ ] `Tail` возвращает кортеж без первого элемента
- [ ] `Reverse` рекурсивно разворачивает кортеж
- [ ] `FlattenOnce` раскрывает один уровень вложенных массивов
- [ ] `Zip` попарно объединяет элементы двух кортежей
- [ ] Длина кортежа вычисляется на уровне типов

## Как проверить себя

1. Нажмите кнопку «Запустить»
2. Убедитесь, что `Head<[string, number]>` дает `string`, а `Last` дает `number`
3. Убедитесь, что `Reverse<[1, 2, 3]>` дает `[3, 2, 1]`
4. Проверьте, что `Zip` корректно работает с кортежами одинаковой длины
