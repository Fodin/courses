# Task 6.2: The use client Directive

## Objective

Understand when and why to add the `'use client'` directive.

## Requirements

1. Show code examples of components
2. The user decides whether the `'use client'` directive is needed
3. Explain the rules: useState/useEffect → client, event handlers → client
4. Show that server components do not need the directive

## Checklist

- [ ] Component examples with and without `'use client'`
- [ ] Interactive selection for each example
- [ ] Rule explanations
- [ ] Visual highlighting of the directive in code

## How to verify

1. For each example, determine whether the directive is needed
2. Check your answer against the hints
