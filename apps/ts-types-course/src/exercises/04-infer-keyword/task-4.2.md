# Задание 4.2: Infer в параметрах функции

## Цель

Научиться извлекать типы параметров из функций и конструкторов с помощью `infer`.

## Требования

1. Создайте тип `FirstParam<T>`, извлекающий тип первого параметра функции
2. Создайте тип `MyParameters<T>`, аналог встроенного `Parameters` — возвращает кортеж параметров
3. Создайте тип `LastParam<T>`, извлекающий тип последнего параметра (используйте variadic tuples: `[...infer _, infer L]`)
4. Создайте тип `ConstructorParams<T>`, извлекающий параметры конструктора класса (обратите внимание на `new` в сигнатуре)
5. Продемонстрируйте извлечение типа `this` из метода с явной аннотацией `this`

## Чеклист

- [ ] `FirstParam` корректно извлекает первый параметр
- [ ] `MyParameters` возвращает кортеж всех параметров
- [ ] `LastParam` извлекает последний параметр через `[...infer _, infer L]`
- [ ] `ConstructorParams` работает с классами через `new (...args: infer P)`
- [ ] Все примеры отображают правильные результаты на странице

## Как проверить себя

1. Нажмите кнопку «Запустить»
2. Для функции `greet(name: string, age: number)` убедитесь, что `FirstParam` дает `string`, а `LastParam` дает `number`
3. Убедитесь, что `ConstructorParams` корректно извлекает типы из конструктора класса
