# Задание 3.1: Buffer Basics

## Цель

Научиться создавать Buffer, конвертировать между кодировками и понимать разницу между длиной строки и размером буфера.

## Требования

1. Продемонстрируйте создание Buffer через from, alloc, allocUnsafe
2. Покажите конвертацию между UTF-8, hex и base64
3. Покажите разницу byte length vs string length для ASCII, кириллицы и emoji
4. Перечислите основные методы Buffer API
5. Реализуйте кнопку для запуска демонстрации

## Чеклист

- [ ] Показаны все способы создания Buffer
- [ ] Конвертация UTF-8 ↔ hex ↔ base64
- [ ] Разница byte vs string length для разных алфавитов
- [ ] Основные методы API перечислены
- [ ] Компонент использует useState

## Как проверить себя

1. Нажмите "Запустить" и изучите размеры буферов
2. Проверьте, что кириллица занимает 2 байта на символ
3. Убедитесь, что emoji занимает 4 байта
