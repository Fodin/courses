# Задание 5.4: useDeferredValue initialValue

## Цель

Использовать новый параметр `initialValue` у `useDeferredValue` (React 19).

## Требования

1. Создайте поле поиска с фильтрацией списка
2. Используйте `useDeferredValue(query, '')` — второй аргумент это initialValue
3. При первом рендере используется initialValue (пустая строка)
4. Покажите, что фильтрация не блокирует ввод

## Чеклист

- [ ] `useDeferredValue` используется с двумя аргументами
- [ ] `initialValue` задаёт начальное значение
- [ ] Ввод в поле поиска остаётся отзывчивым
- [ ] Фильтрация списка обновляется с задержкой

## Как проверить себя

1. Введите текст в поле поиска — поле реагирует мгновенно
2. Список фильтруется с небольшой задержкой
