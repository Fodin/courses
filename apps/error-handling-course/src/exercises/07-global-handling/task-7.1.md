# Задание 7.1: window.onerror

## Цель
Настроить глобальные обработчики ошибок для перехвата необработанных ошибок и промисов.

## Требования
1. Подпишитесь на `window.addEventListener('error', ...)` в `useEffect`
2. Подпишитесь на `unhandledrejection` для необработанных промисов
3. Создайте кнопку, бросающую ошибку в `setTimeout`
4. Создайте кнопку, создающую `Promise.reject` без catch
5. Отобразите перехваченные события на странице
6. Не забудьте отписаться в cleanup

## Чеклист
- [ ] Обработчик `error` event
- [ ] Обработчик `unhandledrejection`
- [ ] Глобальная ошибка перехватывается
- [ ] Необработанный промис перехватывается
- [ ] Cleanup в useEffect
