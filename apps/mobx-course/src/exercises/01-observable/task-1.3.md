# Задание 1.3: Типы observable

## Цель
Работа с разными типами коллекций: массив, Map, Set.

## Требования

- [ ] Создайте `CollectionStore` с `items: string[]`, `tags: Map`, `selectedIds: Set`
- [ ] Добавьте методы `addItem`, `removeItem` для массива
- [ ] Добавьте методы `setTag`, `removeTag` для Map
- [ ] Добавьте метод `toggleId` для Set
- [ ] Добавьте computed `summary` — краткая статистика
- [ ] Покажите все коллекции и кнопки управления в компоненте
